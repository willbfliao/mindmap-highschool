# 平面運動

## 向量運算
### 向量基本性質
- 向量（vector）: 具有大小與方向的物理量
- 純量（scalar）: 只有大小的物理量
- 向量表示法: 箭頭符號或粗體字母
### 向量加減法
- 平行四邊形法: 兩向量共起點，對角線為合向量
- 三角形法（頭尾相接法）: 第一向量尾接第二向量頭
- 向量減法: A - B = A + (-B)
### 向量分解
- 正交分解: 將向量分解為 x、y 分量
  - Aₓ = A cosθ
  - Aᵧ = A sinθ
- 合成: A = √(Aₓ² + Aᵧ²)、tanθ = Aᵧ / Aₓ

## 拋體運動
### 水平拋射
- 水平方向: 等速運動，x = v₀t
- 鉛直方向: 自由落體，y = ½gt²
- 合速度: v = √(v₀² + (gt)²)
- 軌跡: 拋物線（y = gx² / 2v₀²）
### 斜向拋射
- 水平分量: vₓ = v₀cosθ（等速）
- 鉛直分量: vᵧ = v₀sinθ - gt（等加速度）
- 水平距離: x = v₀cosθ · t
- 鉛直高度: y = v₀sinθ · t - ½gt²
### 斜拋重要結論
- 最大高度: H = v₀²sin²θ / (2g)
- 滯空時間: T = 2v₀sinθ / g
- 水平射程: R = v₀²sin2θ / g
- 最大射程: θ = 45° 時射程最遠
- 互補角（如 30° 與 60°）射程相同

## 相對運動
- 相對速度: v_AB = v_A - v_B（向量相減）
- 物體 A 相對於 B 的速度 = A 的速度 - B 的速度
- 應用: 渡河問題（船速 + 水流速的向量合成）
