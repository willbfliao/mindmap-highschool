# 功與能量

## 變力作功
### 功的一般定義
- 恆力作功: W = F·d·cosθ
- 變力作功: W = ∫F·dr（F-x 圖下面積）
- 圖解法: F-x 圖中曲線下的面積即為功
### 彈力作功
- 彈簧力: F = -kx（虎克定律）
- 彈力作功: W = ½kx₁² - ½kx₂²
- 從自然長度拉伸 x: W = -½kx²（外力做正功 ½kx²）

## 重力位能
- 重力位能（gravitational PE）: U = mgh
- 位能零點可任意選取
- 重力作功 = -ΔU = -(U_末 - U_初) = mgh₁ - mgh₂
- 重力為保守力: 做功僅與起終點有關，與路徑無關

## 彈力位能
- 彈性位能（elastic PE）: U = ½kx²
- x: 彈簧伸長量或壓縮量（相對自然長度）
- 彈力作功 = -ΔU = ½kx₁² - ½kx₂²
- 彈力也是保守力

## 力學能守恆
### 守恆條件
- 系統僅受保守力作用（重力、彈力）
- KE₁ + U₁ = KE₂ + U₂
- ½mv₁² + mgh₁ = ½mv₂² + mgh₂
### 含彈簧系統
- ½mv₁² + mgh₁ + ½kx₁² = ½mv₂² + mgh₂ + ½kx₂²
### 非保守力做功
- 有摩擦力時: KE₁ + U₁ + W_非保守 = KE₂ + U₂
- 摩擦力做負功: W_f = -fₖd（力學能減少，轉化為熱能）

## 功率
- 功率（power）: P = W / t = F·v
- 瞬時功率: P = Fv cosθ
- 單位: 瓦特（W），1 hp ≈ 746 W
- 等速行駛: 引擎功率 = 摩擦力 × 速度
