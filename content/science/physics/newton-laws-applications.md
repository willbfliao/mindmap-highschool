# 牛頓運動定律應用

## 摩擦力進階
### 靜摩擦力
- 靜摩擦力（static friction）: 0 ≤ fs ≤ μsN
- 最大靜摩擦力: fs,max = μsN
- 靜摩擦力隨外力增大而增大，直到最大值
- 物體在斜面上不滑動條件: tanθ ≤ μs
### 動摩擦力
- 動摩擦力（kinetic friction）: fk = μkN
- μk < μs（動摩擦係數小於靜摩擦係數）
- 與接觸面積無關，與速度大小無關（理想狀態）
### 摩擦力的應用
- 斜面問題: 上滑 a = g(sinθ + μkcosθ)、下滑 a = g(sinθ - μkcosθ)
- 車輛煞車: 煞車距離 d = v₀² / (2μkg)
- 傳動帶: 靜摩擦力驅動物體

## 等速率圓周運動
### 基本物理量
- 角速度（angular velocity）: ω = 2π / T = 2πf
  - 單位: rad/s
- 週期（period）: T = 2πr / v（繞一圈的時間）
- 頻率（frequency）: f = 1 / T（每秒繞圈數）
### 向心加速度
- 向心加速度（centripetal acceleration）: ac = v² / r = ω²r
- 方向: 永遠指向圓心
- 速率不變但速度方向持續改變 → 有加速度
### 向心力
- 向心力（centripetal force）: Fc = mv² / r = mω²r
- 向心力不是新的力，是合力的效果
- 來源: 繩張力、重力、摩擦力、正向力等

## 圓周運動的應用
### 水平圓周運動
- 汽車過彎: 摩擦力提供向心力，v_max = √(μsgr)
- 傾斜路面: 正向力水平分量提供向心力，tanθ = v² / (rg)
- 圓錐擺: T sinθ = mv² / r、T cosθ = mg
### 鉛直圓周運動
- 最高點: mg - N = mv² / r（繩拉力或軌道正向力最小）
- 最低點: N - mg = mv² / r（繩拉力或軌道正向力最大）
- 最高點不脫離條件: v ≥ √(gr)
- 最低點最小速度: v ≥ √(5gr)（完整繞圈）

## 非慣性座標系
- 慣性座標系: 牛頓定律適用（靜止或等速運動的參考系）
- 非慣性座標系: 需引入假想力（加速運動的參考系）
- 假想力（pseudo force）: F = -ma（方向與加速度相反）
- 電梯問題: 上升加速時體重計讀數增加（N = m(g+a)）
