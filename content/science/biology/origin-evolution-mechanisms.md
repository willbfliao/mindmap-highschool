# 生命的起源與演化機制

## 生命的起源

### 原始地球環境
- 約 **46 億年前**地球形成
- 原始大氣：主要含 **H₂O、CO₂、N₂、H₂S**，幾乎**無 O₂**（還原性大氣）
- 充滿閃電、紫外線、火山活動等**能量來源**
- 海洋逐漸冷凝形成

### 化學演化（chemical evolution）
- **米勒-尤里實驗（Miller-Urey, 1953）**
  - 模擬原始地球大氣條件
  - 混合 CH₄、NH₃、H₂、H₂O + 放電
  - 成功合成**胺基酸**等有機小分子
  - 證明無機物可在自然條件下合成有機物

### 從簡單到複雜
- **有機小分子** → **有機大分子**（蛋白質、核酸等）
- **RNA 世界假說（RNA world hypothesis）**：
  - 早期生命可能以 **RNA** 同時擔任遺傳物質與催化劑（核糖酵素 ribozyme）
  - 後來 DNA 取代 RNA 成為遺傳物質（更穩定）
- **原始細胞（protocell）**：脂質雙層自組裝形成囊泡，包覆自我複製的分子
- 最早的生命形式可能出現於約 **38～40 億年前**

### 早期生命的演化
- 最早的生物為**原核生物**（類似厭氧細菌）
- **藍綠菌（cyanobacteria）**出現 → 行光合作用 → 產生 **O₂**
- 大氧化事件（Great Oxidation Event, ~24 億年前）→ 大氣中 O₂ 累積
- 好氧生物演化出現
- **內共生學說**：真核細胞的粒線體與葉綠體源自被吞噬的原核生物

## 演化的機制

### 天擇（natural selection）
- 自然環境對族群中不同表現型個體施加**選擇壓力**
- 具有**有利性狀**的個體較能存活與繁殖
- 有利等位基因的頻率在族群中**逐代增加**
- 天擇是唯一具有**適應性方向**的演化力量

### 性擇（sexual selection）
- 影響個體**交配成功率**的選擇
- **同性競爭（intrasexual selection）**：同性之間為爭奪交配權而競爭（如雄鹿角鬥）
- **異性選擇（intersexual selection）**：一性選擇具有特定性狀的另一性（如孔雀尾羽）

### 遺傳漂變（genetic drift）
- 等位基因頻率的**隨機變化**，在**小族群**中影響最大
- **瓶頸效應（bottleneck effect）**：族群數量因災難大幅減少 → 存活者的基因組成偏離原族群
- **建立者效應（founder effect）**：少數個體遷移到新地區建立族群 → 基因頻率與母族群不同
- 漂變為**非適應性**的，可導致有害等位基因的固定

### 基因流動（gene flow）
- 個體在族群間**遷移**，攜帶等位基因進入或離開族群
- 增加族群間的基因交流 → 減少族群間的遺傳差異
- 減少基因流動 → 促進族群分化

### 基因突變（mutation）
- 產生**新的等位基因**，為演化提供原始材料
- 突變為**隨機**發生，大多為中性或有害
- 只有少數突變在特定環境中具有適應性優勢

## 哈地-溫伯格平衡（Hardy-Weinberg equilibrium）

### 平衡條件
- 在理想族群中，等位基因頻率**世代不變**
- 五大假設：大族群、無突變、無遷移、隨機交配、無天擇
- 現實中很少族群完全符合 → 族群持續演化

### 數學公式
- p + q = 1（p = 顯性等位基因頻率，q = 隱性頻率）
- p² + 2pq + q² = 1
- 應用：已知 q²（隱性表現型頻率）→ 反推 q → 計算 p → 得出攜帶者頻率（2pq）

## 物種形成（speciation）

### 物種的定義
- **生物學種概念**：能在自然狀態下**交配並產生具有生殖能力後代**的群體
- **生殖隔離（reproductive isolation）**：不同物種間無法成功交配或產生可育後代

### 生殖隔離的機制
#### 合子前隔離（prezygotic barriers）
- **棲地隔離**：生活在不同棲息地
- **時間隔離**：繁殖季節不同
- **行為隔離**：求偶行為不同
- **機械隔離**：生殖器官構造不相容
- **配子隔離**：精卵無法結合

#### 合子後隔離（postzygotic barriers）
- **雜交不活**：雜交胚胎無法正常發育
- **雜交不孕**：雜交後代不具生殖能力（如騾）
- **雜交衰退**：第二代以後適應力降低

### 異域種化（allopatric speciation）
- **地理隔離**（如山脈、海洋）將族群分成兩群
- 各自獨立演化（天擇、漂變）→ 累積遺傳差異
- 最終形成生殖隔離 → 成為不同物種
- 最常見的物種形成模式

### 同域種化（sympatric speciation）
- 同一地理區域內，**無地理隔離**下形成新物種
- 機制：**多倍體（polyploidy）**在植物中很常見
  - **同源多倍體**：同一物種的染色體加倍
  - **異源多倍體**：雜交後染色體加倍（如小麥的演化）
- 也可因**生態區位分化**或**性擇**促進

### 適應輻射（adaptive radiation）
- 一個祖先物種在短時間內分化出**多個物種**
- 通常發生在新的生態環境中（島嶼、大滅絕後）
- 經典例子：
  - 達爾文雀（加拉巴哥群島）：不同島嶼演化出不同喙部形態
  - 哺乳動物在恐龍滅絕後的輻射
