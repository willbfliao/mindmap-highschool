# 消化系統

## 消化道（alimentary canal）

### 口腔（oral cavity）
- 機械性消化：牙齒咀嚼將食物磨碎
- 化學性消化：唾液中的**唾液澱粉酶（salivary amylase）**分解澱粉為麥芽糖
- 唾液澱粉酶最適 pH 值約為 **6.8（中性偏酸）**
- 舌頭協助攪拌食物形成**食團（bolus）**

### 咽與食道（pharynx & esophagus）
- 吞嚥時**會厭軟骨（epiglottis）**蓋住氣管，防止食物誤入
- 食道以**蠕動（peristalsis）**方式推送食團至胃
- 食道無消化與吸收功能

### 胃（stomach）
- **胃液**成分：鹽酸（HCl）、胃蛋白酶原（pepsinogen）、黏液
- 鹽酸功能：殺菌、活化胃蛋白酶原為**胃蛋白酶（pepsin）**、提供酸性環境
- 胃蛋白酶最適 pH 值約為 **1.5～2.0**
- 胃蛋白酶將蛋白質分解為**多肽（polypeptides）**
- **黏液**保護胃壁不被胃酸侵蝕
- 食物在胃中混合形成**食糜（chyme）**

### 小腸（small intestine）
#### 結構特點
- 全長約 **6～7 公尺**，為消化道中最長的部分
- 分為**十二指腸（duodenum）**、空腸（jejunum）、迴腸（ileum）
- 十二指腸接收胰液與膽汁

#### 消化功能
- 為化學消化與吸收的**主要場所**
- 小腸液含多種消化酵素完成最終消化
- 腸液 pH 值約為 **7.5～8.5（弱鹼性）**

#### 小腸絨毛構造（intestinal villi）
- 環狀皺壁 → 絨毛 → 微絨毛（刷狀緣），大幅增加**表面積**
- 每根絨毛內含：
  - **微血管網**：吸收醣類、胺基酸、水溶性維生素、礦物質
  - **乳糜管（lacteal）**：吸收脂肪酸與甘油重新合成的脂肪，進入淋巴系統
- 絨毛上皮為**單層柱狀上皮細胞**，利於物質通透

### 大腸（large intestine）
- 主要功能：吸收**水分**與**電解質**
- 含大量**腸道菌群**，可合成維生素 B 群與維生素 K
- 食物殘渣形成**糞便**，經直腸與肛門排出
- 大腸無絨毛構造，不進行營養素消化

## 消化腺（digestive glands）

### 唾液腺（salivary glands）
- 三對主要唾液腺：腮腺、頜下腺、舌下腺
- 分泌**唾液**：含唾液澱粉酶與黏液
- 唾液澱粉酶：澱粉 → 麥芽糖

### 肝臟與膽囊（liver & gallbladder）
- 肝臟為人體**最大的內臟器官**
- 肝臟分泌**膽汁（bile）**，儲存於膽囊
- 膽汁**不含消化酵素**
- 膽汁功能：**乳化脂肪（emulsification）**，將大脂肪滴分散為小脂肪滴，增加脂肪酶作用面積
- 膽汁經由總膽管注入十二指腸

### 胰臟（pancreas）
- 為**外分泌**與**內分泌**雙重功能腺體
- 外分泌部分泌**胰液**注入十二指腸
- 胰液含重要消化酵素：
  - **胰澱粉酶**：澱粉 → 麥芽糖
  - **胰蛋白酶（trypsin）**：蛋白質 → 多肽 → 胺基酸
  - **胰脂肪酶（lipase）**：脂肪 → 脂肪酸 + 甘油
- 胰液含**碳酸氫鈉（NaHCO₃）**，中和胃酸，提供鹼性環境

## 酵素消化總整理

### 醣類消化
- 口腔：唾液澱粉酶 → 澱粉分解為**麥芽糖**
- 小腸：胰澱粉酶 → 澱粉分解為麥芽糖
- 小腸：麥芽糖酶 → 麥芽糖分解為**葡萄糖**
- 最終產物：**單醣（葡萄糖、果糖、半乳糖）**

### 蛋白質消化
- 胃：胃蛋白酶 → 蛋白質分解為**多肽**
- 小腸：胰蛋白酶 → 多肽進一步分解
- 小腸：肽酶 → 多肽分解為**胺基酸**
- 最終產物：**胺基酸（amino acids）**

### 脂肪消化
- 小腸：膽汁**乳化**脂肪（物理性）
- 小腸：胰脂肪酶 → 脂肪分解為**脂肪酸 + 甘油**
- 最終產物：**脂肪酸（fatty acids）與甘油（glycerol）**

## 營養素吸收

### 吸收部位
- **小腸**為主要吸收部位（約 90% 以上的營養素）
- 胃：少量吸收水、酒精、部分藥物
- 大腸：吸收水分與電解質

### 吸收方式
- **主動運輸（active transport）**：葡萄糖、胺基酸（逆濃度梯度，需消耗 ATP）
- **被動擴散（passive diffusion）**：脂肪酸、甘油、水（順濃度梯度）
- **促進擴散（facilitated diffusion）**：果糖

### 吸收路徑
- 水溶性營養素（醣類、胺基酸）→ 微血管 → **肝門靜脈** → 肝臟處理
- 脂溶性營養素（脂肪酸、甘油）→ 重新合成脂肪 → 乳糜管 → **淋巴系統** → 血液循環
