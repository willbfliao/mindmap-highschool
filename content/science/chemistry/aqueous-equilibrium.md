# 水溶液中的平衡

## 水的自解離
### 水的自解離反應
- 純水中水分子間可發生質子轉移
  - $2\text{H}_2\text{O} \rightleftharpoons \text{H}_3\text{O}^+ + \text{OH}^-$
  - 簡寫為 $\text{H}_2\text{O} \rightleftharpoons \text{H}^+ + \text{OH}^-$
- 水的離子積常數 $K_w$
  - $K_w = [\text{H}^+][\text{OH}^-] = 1.0 \times 10^{-14}$（25°C）
  - 純水中 $[\text{H}^+] = [\text{OH}^-] = 1.0 \times 10^{-7} \text{ M}$
  - $K_w$ 隨溫度升高而增大（自解離為吸熱反應）
### pH 值與 pOH 值
- $\text{pH} = -\log[\text{H}^+]$，$\text{pOH} = -\log[\text{OH}^-]$
- 25°C 時 $\text{pH} + \text{pOH} = 14$
- 酸性溶液：$\text{pH} < 7$；中性：$\text{pH} = 7$；鹼性：$\text{pH} > 7$
  - 💡 聯想：pH 像溫度計——數字越小越「酸」，越大越「鹼」，7 是中間分界線

## 弱酸弱鹼的游離平衡
### 弱酸的游離平衡
- 弱酸 $\text{HA}$ 在水中部分游離：$\text{HA} \rightleftharpoons \text{H}^+ + \text{A}^-$
- 酸的游離常數 $K_a = \frac{[\text{H}^+][\text{A}^-]}{[\text{HA}]}$
- $K_a$ 越大，酸性越強
  - 如醋酸 $K_a = 1.8 \times 10^{-5}$，氫氟酸 $K_a = 6.8 \times 10^{-4}$
### 弱鹼的游離平衡
- 弱鹼 $\text{B}$ 在水中部分游離：$\text{B} + \text{H}_2\text{O} \rightleftharpoons \text{BH}^+ + \text{OH}^-$
- 鹼的游離常數 $K_b = \frac{[\text{BH}^+][\text{OH}^-]}{[\text{B}]}$
- $K_b$ 越大，鹼性越強
  - 如氨 $K_b = 1.8 \times 10^{-5}$
### 游離度
- 游離度 $\alpha = \frac{\text{已游離的莫耳數}}{\text{初始莫耳數}} \times 100\%$
- 稀釋會使游離度增大（但 $[\text{H}^+]$ 不一定增大）
- 💡 聯想：游離度像出席率——濃度越稀，「出來游離」的比例越高
### 共軛酸鹼對的關係
- $K_a \times K_b = K_w$（共軛酸鹼對）
- 弱酸的 $K_a$ 越小，其共軛鹼的 $K_b$ 越大
- 可用於判斷鹽類水溶液的酸鹼性

## 同離子效應
### 定義與原理
- 在弱電解質的平衡溶液中加入含有相同離子的強電解質
- 平衡向減少該離子的方向移動（勒沙特列原理）
- 使弱電解質的游離度降低
### 應用範例
- 醋酸溶液中加入醋酸鈉（$\text{CH}_3\text{COONa}$）
  - $\text{CH}_3\text{COOH} \rightleftharpoons \text{H}^+ + \text{CH}_3\text{COO}^-$
  - 加入 $\text{CH}_3\text{COO}^-$ 使平衡左移，$[\text{H}^+]$ 降低
- 氨水中加入氯化銨（$\text{NH}_4\text{Cl}$）
  - 加入 $\text{NH}_4^+$ 使氨的游離平衡左移，$[\text{OH}^-]$ 降低
  - 💡 聯想：像在擁擠的教室再塞進同學——空間不夠，部分人只好退回去

## 緩衝溶液
### 緩衝溶液的組成
- 由弱酸與其共軛鹼（或弱鹼與其共軛酸）的混合溶液組成
  - 如 $\text{CH}_3\text{COOH}$ / $\text{CH}_3\text{COONa}$（弱酸＋共軛鹼）
  - 如 $\text{NH}_3$ / $\text{NH}_4\text{Cl}$（弱鹼＋共軛酸）
### 緩衝作用原理
- 加入少量酸時：共軛鹼中和 $\text{H}^+$，pH 變化小
  - $\text{A}^- + \text{H}^+ \rightarrow \text{HA}$
- 加入少量鹼時：弱酸中和 $\text{OH}^-$，pH 變化小
  - $\text{HA} + \text{OH}^- \rightarrow \text{A}^- + \text{H}_2\text{O}$
- 緩衝容量有限，加入過多酸或鹼將超出緩衝範圍
### Henderson-Hasselbalch 方程式
- $\text{pH} = \text{p}K_a + \log\frac{[\text{A}^-]}{[\text{HA}]}$
- 當 $[\text{A}^-] = [\text{HA}]$ 時，$\text{pH} = \text{p}K_a$
- 緩衝範圍約為 $\text{p}K_a \pm 1$
  - 💡 聯想：Henderson-Hasselbalch 方程式像天秤——$[\text{A}^-]$ 和 $[\text{HA}]$ 的比值決定 pH 偏向哪邊
### 生物體中的緩衝系統
- 血液中的碳酸—碳酸氫鹽緩衝系統：$\text{H}_2\text{CO}_3$ / $\text{HCO}_3^-$
- 維持血液 pH 在 7.35–7.45 的狹窄範圍

## 酸鹼滴定曲線
### 強酸滴定強鹼
- 當量點 $\text{pH} = 7$（中性）
- 曲線在當量點附近 pH 急劇變化
- 適用指示劑範圍較廣（如酚酞、甲基橙皆可）
### 強鹼滴定弱酸
- 當量點 $\text{pH} > 7$（因共軛鹼水解呈鹼性）
- 半當量點時 $\text{pH} = \text{p}K_a$（緩衝區間）
- 適用指示劑：酚酞（變色範圍 pH 8.2–10.0）
### 強酸滴定弱鹼
- 當量點 $\text{pH} < 7$（因共軛酸水解呈酸性）
- 適用指示劑：甲基橙（變色範圍 pH 3.1–4.4）
### 指示劑的選擇原則
- 指示劑變色範圍應涵蓋當量點 pH
- 常用指示劑及變色範圍
  - 甲基橙：pH 3.1–4.4（紅→橙→黃）
  - 石蕊：pH 5.0–8.0（紅→紫→藍）
  - 酚酞：pH 8.2–10.0（無色→粉紅）
  - 💡 聯想：選指示劑像選衣服——要在「對的場合」（當量點 pH）變色才有意義

## 沉澱與溶解度積常數
### 溶解度積常數（$K_{sp}$）
- 難溶鹽在飽和溶液中的離子濃度乘積
- 對 $\text{A}_m\text{B}_n \rightleftharpoons m\text{A}^{n+} + n\text{B}^{m-}$
  - $K_{sp} = [\text{A}^{n+}]^m[\text{B}^{m-}]^n$
- $K_{sp}$ 越小，溶解度越低
  - 如 $\text{AgCl}$：$K_{sp} = 1.8 \times 10^{-10}$
  - 如 $\text{BaSO}_4$：$K_{sp} = 1.1 \times 10^{-10}$
### 沉澱的生成與溶解判斷
- 離子積 $Q_{sp}$ 與 $K_{sp}$ 的比較
  - $Q_{sp} > K_{sp}$：產生沉澱
  - $Q_{sp} = K_{sp}$：飽和溶液（平衡）
  - $Q_{sp} < K_{sp}$：未飽和，沉澱可溶解
### 共同離子效應對溶解度的影響
- 加入含有共同離子的溶液會降低難溶鹽的溶解度
  - 如在 $\text{AgCl}$ 飽和溶液中加入 $\text{NaCl}$
  - $[\text{Cl}^-]$ 增大 → $Q_{sp} > K_{sp}$ → 產生更多沉澱
  - 💡 聯想：像游泳池已經滿了，再加水（共同離子）就會溢出來（沉澱）
### 選擇性沉澱
- 利用不同鹽類 $K_{sp}$ 的差異，逐步加入沉澱劑
- $K_{sp}$ 較小者先沉澱
- 可用於分離混合溶液中的不同離子
### 溶解沉澱的方法
- 加酸溶解（如 $\text{CaCO}_3 + \text{HCl}$）
- 形成錯合物（如 $\text{AgCl} + \text{NH}_3 \rightarrow [\text{Ag}(\text{NH}_3)_2]^+$）
- 氧化還原法溶解
