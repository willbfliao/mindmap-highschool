# 循環系統

## 心臟
### 構造
- 位於胸腔中央偏左，約拳頭大小
- 四個腔室
  - 左心房: 接收肺靜脈的含氧血
  - 左心室: 將含氧血泵入主動脈（體循環）
  - 右心房: 接收上下腔靜脈的缺氧血
  - 右心室: 將缺氧血泵入肺動脈（肺循環）
- 左心室壁最厚（需將血液泵至全身）
- 心臟瓣膜（防止血液逆流）
  - 房室瓣
    - 二尖瓣（僧帽瓣）: 左心房與左心室之間
    - 三尖瓣: 右心房與右心室之間
  - 半月瓣
    - 主動脈瓣: 左心室與主動脈之間
    - 肺動脈瓣: 右心室與肺動脈之間
### 心臟的傳導系統
- 竇房結（SA node）: 位於右心房，為心律調節器（pacemaker）
- 房室結（AV node）: 位於心房與心室交界處
- 乏氏束（bundle of His）: 將衝動傳至心室
- 乏金乏纖維（Purkinje fibers）: 分布於心室壁，引發心室收縮
- 傳導順序: 竇房結 → 房室結 → 乏氏束 → 乏金乏纖維 → 心室收縮
### 心動週期
- 心房收縮期: 血液由心房壓入心室
- 心室收縮期: 房室瓣關閉，血液泵入動脈
- 心室舒張期: 半月瓣關閉，血液由靜脈回流心房
- 正常心率: 約 60-100 次/分鐘
### 心臟的調節
- 自律性: 竇房結自動產生節律性電衝動
- 交感神經: 加速心跳
- 副交感神經（迷走神經）: 減慢心跳
- 腎上腺素: 加速心跳、增強收縮力

## 血管
### 動脈（artery）
- 管壁厚、彈性大、有平滑肌
- 承受高壓，將血液從心臟送往全身
- 血流速度快
- 無瓣膜（靠血壓推動血流）
- 大動脈 → 中動脈 → 小動脈 → 微動脈
### 靜脈（vein）
- 管壁薄、彈性小
- 血壓低，將血液從全身送回心臟
- 含瓣膜: 防止血液逆流
- 血液回流的輔助機制
  - 骨骼肌收縮擠壓
  - 呼吸時胸腔負壓
  - 靜脈瓣膜
### 微血管（capillary）
- 管壁最薄（僅一層內皮細胞）
- 管徑最小（僅容一個紅血球通過）
- 血流速度最慢
- 為物質交換的場所（O₂、CO₂、養分、廢物）
- 分布最廣，遍布全身組織
### 血管比較
- 管壁厚度: 動脈 > 靜脈 > 微血管
- 管腔大小: 靜脈 > 動脈 > 微血管
- 血流速度: 動脈 > 靜脈 > 微血管
- 總截面積: 微血管 > 靜脈 > 動脈

## 血液
### 血液成分
#### 血漿（約佔 55%）
- 水: 約佔血漿 90%
- 血漿蛋白
  - 白蛋白（albumin）: 維持血液滲透壓
  - 球蛋白（globulin）: 包含抗體（免疫球蛋白）
  - 纖維蛋白原（fibrinogen）: 參與凝血
- 養分: 葡萄糖、胺基酸、脂質
- 廢物: 尿素、尿酸
- 電解質: Na⁺、K⁺、Ca²⁺、Cl⁻
- 激素、氣體（少量 O₂、CO₂）
#### 血球（約佔 45%）
##### 紅血球（erythrocyte / RBC）
- 雙凹圓盤狀，無細胞核（成熟時）
- 含血紅素（hemoglobin / Hb）: 運送 O₂ 與少量 CO₂
- 由骨髓製造，壽命約 120 天
- 老化紅血球在肝臟與脾臟中被分解
- 正常數量: 男性約 500 萬/μL，女性約 450 萬/μL
- 紅血球過少或血紅素不足: 貧血
##### 白血球（leukocyte / WBC）
- 有細胞核，可做變形運動
- 正常數量: 約 5000-10000/μL
- 功能: 免疫防禦
- 種類
  - 嗜中性球: 數量最多，吞噬細菌
  - 嗜酸性球: 對抗寄生蟲、參與過敏反應
  - 嗜鹼性球: 釋放組織胺，參與發炎與過敏
  - 單核球: 進入組織後分化為巨噬細胞
  - 淋巴球: B 細胞與 T 細胞，參與專一性免疫
##### 血小板（platelet / thrombocyte）
- 由巨核細胞碎片形成，無細胞核
- 正常數量: 約 15-40 萬/μL
- 功能: 凝血與止血

### 血型
#### ABO 血型系統
- 由紅血球表面抗原（凝集原）決定
  - A 型: 有 A 抗原，血漿含抗 B 抗體
  - B 型: 有 B 抗原，血漿含抗 A 抗體
  - AB 型: 有 A 和 B 抗原，無抗體（萬能受血者）
  - O 型: 無抗原，血漿含抗 A 和抗 B 抗體（萬能捐血者）
- 輸血原則: 避免捐血者紅血球抗原與受血者血漿抗體結合（造成凝集反應）
- 遺傳: I^A、I^B 為共顯性，i 為隱性
  - A 型: I^A I^A 或 I^A i
  - B 型: I^B I^B 或 I^B i
  - AB 型: I^A I^B
  - O 型: ii
#### Rh 血型系統
- Rh⁺: 紅血球有 Rh 抗原（大多數人）
- Rh⁻: 紅血球無 Rh 抗原
- 新生兒溶血症: Rh⁻ 母親懷第二胎 Rh⁺ 胎兒時可能發生

### 凝血機制
- 血管損傷 → 血小板聚集形成血小板栓子
- 凝血因子連鎖反應（凝血瀑布）
  - 凝血酶原（prothrombin）→ 凝血酶（thrombin）
  - 纖維蛋白原（fibrinogen）→ 纖維蛋白（fibrin）
- 纖維蛋白形成纖維網 → 捕捉血球 → 形成血塊
- 需要 Ca²⁺ 與維生素 K 參與
- 抗凝血機制: 肝素（heparin）、纖維蛋白溶解酶

## 循環路徑
### 體循環（大循環）
- 路徑: 左心室 → 主動脈 → 全身動脈 → 微血管（物質交換）→ 全身靜脈 → 上下腔靜脈 → 右心房
- 功能: 將含氧血送至全身組織，帶回缺氧血
- 特徵: 動脈攜帶含氧血，靜脈攜帶缺氧血
### 肺循環（小循環）
- 路徑: 右心室 → 肺動脈 → 肺部微血管（氣體交換）→ 肺靜脈 → 左心房
- 功能: 將缺氧血送至肺部進行氣體交換，帶回含氧血
- 特徵: 動脈攜帶缺氧血，靜脈攜帶含氧血（與體循環相反）
### 冠狀循環
- 冠狀動脈: 從主動脈根部分出，供應心肌氧氣與養分
- 冠狀靜脈: 匯入冠狀竇，回流至右心房
- 冠狀動脈阻塞: 心肌缺血 → 心絞痛或心肌梗塞

## 淋巴循環
### 淋巴系統的組成
- 淋巴管: 遍布全身，匯集組織液
- 淋巴結: 過濾淋巴液，含大量淋巴球
- 淋巴器官: 脾臟、胸腺、扁桃腺
### 淋巴液
- 來源: 微血管滲出的組織液進入淋巴管
- 成分: 與血漿相似，但蛋白質含量較低
- 含有淋巴球
### 淋巴循環路徑
- 組織液 → 淋巴微管 → 淋巴管 → 淋巴幹 → 左右淋巴管 → 匯入鎖骨下靜脈 → 回到血液循環
- 胸管（左淋巴管）: 收集身體大部分區域的淋巴液
- 右淋巴管: 收集右上半身的淋巴液
### 淋巴系統的功能
- 回收組織液: 維持組織液與血漿間的平衡
- 運輸脂肪: 小腸絨毛中的乳糜管吸收脂溶性養分
- 免疫防禦: 淋巴結中的淋巴球過濾病原體

## 血壓調節
### 血壓的定義
- 血液對血管壁的壓力
- 收縮壓（systolic）: 心室收縮時的最高壓力（正常 < 120 mmHg）
- 舒張壓（diastolic）: 心室舒張時的最低壓力（正常 < 80 mmHg）
### 影響血壓的因素
- 心輸出量: 心跳速率 × 每搏輸出量
- 血管阻力: 血管口徑（小動脈收縮/擴張）
- 血液量: 血量增加則血壓升高
- 血液黏稠度: 黏稠度增加則血壓升高
### 血壓調節機制
#### 神經調節（快速）
- 壓力感受器: 位於頸動脈竇與主動脈弓
- 血壓升高 → 壓力感受器興奮 → 延腦心血管中樞 → 副交感神經興奮 → 心跳減慢、血管擴張 → 血壓下降
- 血壓降低 → 交感神經興奮 → 心跳加快、血管收縮 → 血壓升高
#### 激素調節（較慢）
- 腎素—血管張力素—醛固酮系統（RAA 系統）
  - 血壓降低 → 腎臟分泌腎素 → 血管張力素 II 生成 → 血管收縮 + 醛固酮分泌增加 → Na⁺ 與水再吸收增加 → 血壓升高
- ADH（抗利尿激素）
  - 促進水分再吸收 → 血量增加 → 血壓升高
- 心房利鈉肽（ANP）
  - 心房壁受牽張時分泌
  - 促進 Na⁺ 與水排出 → 血壓降低
### 高血壓
- 定義: 持續血壓 ≥ 140/90 mmHg
- 風險: 動脈硬化、中風、心臟病、腎臟病
