# 植物對環境的感應

## 植物激素（plant hormones）

### 生長素（auxin / IAA）

#### 功能
- 促進**細胞伸長**（低濃度促進，高濃度抑制）
- 促進根的形成（扦插生根）
- **頂芽優勢（apical dominance）**：頂芽產生的生長素抑制側芽生長
- 促進果實發育（可產生無子果實）
- 參與向光性與向地性

#### 生長素的分布特性
- 由莖頂端向下運輸（**極性運輸**）
- 光照不均時，生長素向**背光側**集中
- 重力作用下，生長素向**下側**集中

### 吉貝素（gibberellin / GA）

#### 功能
- 促進**莖的伸長**（尤其是節間伸長）
- 打破種子**休眠**，促進萌發
- 促進開花（部分植物）
- 刺激花粉管生長
- 促進果實發育

#### 應用
- 噴灑吉貝素使葡萄無子且果粒增大
- 促進矮性植物莖的伸長

### 細胞分裂素（cytokinin）

#### 功能
- 促進**細胞分裂**
- 延緩葉片**老化**（保持葉片綠色）
- 促進**側芽生長**（與生長素拮抗）
- 主要在**根尖**合成，向上運輸

#### 生長素與細胞分裂素的比值
- 生長素/細胞分裂素 **高** → 促進根的分化
- 生長素/細胞分裂素 **低** → 促進芽的分化
- 比值相當 → 促進癒傷組織增殖

### 離層酸（abscisic acid / ABA）

#### 功能
- **抑制生長**，為「逆境激素」
- 促進氣孔**關閉**（乾旱逆境時保護水分）
- 維持種子**休眠**（防止在不利條件下萌發）
- 促進**落葉**與**果實脫落**

### 乙烯（ethylene）

#### 功能
- 唯一的**氣態**植物激素
- 促進**果實成熟**（催熟）
- 促進**落葉、落花、落果**（促進離層形成）
- 抑制莖的伸長
- 「一顆爛蘋果壞了一籮筐」：成熟果實釋放乙烯 → 加速鄰近果實成熟

#### 應用
- 用乙烯（或乙烯利）催熟水果
- 鳳梨用乙烯促進開花

### 植物激素的交互作用
- 植物的生長發育受**多種激素**的協調調控
- **協同作用**：如生長素與吉貝素共同促進果實發育
- **拮抗作用**：如生長素與細胞分裂素對側芽生長的相反效果
- **比值效應**：激素間的比例比絕對濃度更重要

## 向性（tropism）

### 向光性（phototropism）
- 植物莖**向光生長**（正向光性）
- 根**背光生長**（負向光性）
- **機制**：
  - 光照不均 → 生長素向**背光側**集中
  - 莖的背光側生長素濃度較高 → 細胞伸長較快 → 莖向光彎曲
  - 達爾文父子的燕麥芽鞘實驗證實光感受部位在**芽鞘尖端**
- **光受體**：向光素（phototropin）感受藍光

### 向地性（gravitropism）
- 根**向地生長**（正向地性）
- 莖**背地生長**（負向地性）
- **機制**：
  - 根冠的**平衡石細胞**含有**澱粉體**，受重力沉降
  - 觸發訊號 → 生長素重新分布
  - 根的下側生長素濃度高 → **抑制**根細胞伸長 → 根向下彎曲
  - 莖的下側生長素濃度高 → **促進**莖細胞伸長 → 莖向上彎曲
- 根與莖對生長素的**敏感度不同**：根對高濃度生長素敏感（被抑制），莖則被促進

### 向觸性（thigmotropism）
- 植物接觸固體物時的生長反應
- 如藤蔓的**捲鬚**纏繞支架
- 接觸側生長較慢，對側較快 → 捲曲

### 向水性（hydrotropism）
- 根向水源方向生長

## 傾性（nastic movement）

### 與向性的差異
- **向性**：生長方向與刺激方向有關
- **傾性**：反應與刺激方向**無關**，為固定方向的運動

### 感震性（thigmonasty）
- 觸碰引起的快速運動
- **含羞草（Mimosa pudica）**：觸碰後小葉合攏、葉柄下垂
  - 機制：**葉枕（pulvinus）**中的細胞快速失水（K⁺ 外流 → 水隨之外流）→ 膨壓下降
  - 非生長反應，可恢復
- **捕蠅草**：觸碰感覺毛 → 葉片快速閉合（捕蟲）

### 睡眠運動（nyctinasty）
- 部分植物的葉片在白天張開、夜晚合攏
- 受**生物時鐘（circadian rhythm）**調控
- 機制同樣涉及葉枕細胞的膨壓變化

### 開花運動
- 部分花朵在特定時間開放與關閉
- 受溫度與光線影響

## 光敏素（phytochrome）

### 光敏素的特性
- 一種**光受體蛋白**，可偵測光的存在與光質
- 存在兩種可互轉形式：
  - **Pr（紅光吸收型）**：吸收 **660 nm 紅光** → 轉變為 Pfr
  - **Pfr（遠紅光吸收型）**：吸收 **730 nm 遠紅光** → 轉變為 Pr
- **Pfr 為活化型**，觸發生理反應
- 在黑暗中 Pfr 緩慢轉回 Pr

### 光敏素調控的反應
- **種子萌發**：部分種子（如萵苣）需要光照（紅光 → Pfr 增加 → 促進萌發）
- **去黃化反應（de-etiolation）**：黑暗中生長的幼苗（黃化苗）接觸光線後正常發育
- **開花的光週期反應**

## 光週期現象（photoperiodism）

### 概念
- 植物對**日照長度（光週期）**的反應，影響開花時機
- 實際上是偵測**暗期長度**

### 植物的分類
- **短日照植物（short-day plants）**：暗期**長於**臨界暗期時開花
  - 如菊花、聖誕紅、水稻（秋冬開花）
- **長日照植物（long-day plants）**：暗期**短於**臨界暗期時開花
  - 如波菜、蘿蔔、小麥（春夏開花）
- **日中性植物（day-neutral plants）**：開花不受日照長度影響
  - 如番茄、玉米、向日葵

### 暗期中斷實驗
- 在長暗期中給予短暫**閃光（紅光）** → Pr → Pfr → 暗期被「中斷」
- 短日照植物在中斷暗期後**不開花**（暗期不夠長）
- 長日照植物在中斷暗期後**開花**
- 若閃光為遠紅光 → Pfr → Pr → 效果被抵消
- 證明植物測量的是**暗期長度**，而非日照長度

### 春化作用（vernalization）
- 某些植物需要經歷一段**低溫期**才能開花
- 如小麥、冬季穀類
- 確保植物在冬季後的適當時機開花
