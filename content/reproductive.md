# 生殖系統

## 男性生殖系統

### 睪丸（testes）
- 位於**陰囊**中，體外位置維持低於體溫 **2～3°C**，利於精子生成
- 含大量**曲細精管（seminiferous tubules）**，為精子生成場所
- 曲細精管壁含**乙狀細胞（Sertoli cells）**：支持與營養精子發育
- 睪丸間質含**萊氏細胞（Leydig cells）**：分泌**睪固酮（testosterone）**

### 精子生成（spermatogenesis）
- 發生於曲細精管中
- **精原細胞**（2n）→ 有絲分裂增殖 → **初級精母細胞**（2n）
- 初級精母細胞 → 減數分裂 I → **次級精母細胞**（n）
- 次級精母細胞 → 減數分裂 II → **精細胞**（n）
- 精細胞經**變態（分化）**→ **成熟精子**
- 一個初級精母細胞最終產生 **4 個精子**
- 精子於**副睪（epididymis）**中儲存並獲得活動力

### 精子構造
- **頭部**：含細胞核（攜帶遺傳物質）與**頂體（acrosome）**（含酵素，穿透卵子）
- **中段**：含大量**粒線體**，提供運動能量
- **尾部（鞭毛）**：擺動推進精子前進

### 附屬腺體與輸送管道
- **副睪**：儲存精子並使其成熟
- **輸精管**：運送精子至尿道
- **儲精囊**：分泌鹼性液體，含果糖，提供精子能量
- **攝護腺（前列腺）**：分泌鹼性液體，中和陰道酸性環境
- **精液 = 精子 + 附屬腺體分泌液**

### 睪固酮（testosterone）的功能
- 促進**精子生成**
- 促進**第二性徵**發育：聲音低沉、體毛生長、肌肉發達
- 維持生殖器官功能

## 女性生殖系統

### 卵巢（ovaries）
- 左右各一，位於**骨盆腔**內
- 功能：產生**卵子**（生殖功能）與分泌**雌激素、黃體素**（內分泌功能）
- 含大量**濾泡（follicles）**，每個濾泡內含一個卵母細胞

### 卵子生成（oogenesis）
- **卵原細胞**（2n）→ 有絲分裂增殖 → **初級卵母細胞**（2n）
- 初級卵母細胞於胎兒時期即已形成，停留在減數分裂 I 前期
- 青春期後每月排卵時 → 減數分裂 I → **次級卵母細胞**（n）+ 第一極體
- 受精時 → 減數分裂 II → **卵子**（n）+ 第二極體
- 一個初級卵母細胞最終產生 **1 個卵子**與 3 個極體（極體退化）

### 輸卵管（oviducts / fallopian tubes）
- 輸卵管**繖部（fimbriae）**承接排出的卵子
- 為正常**受精**發生的場所
- 以纖毛擺動與蠕動將受精卵運送至子宮

### 子宮（uterus）
- 為胚胎著床與發育的場所
- 子宮內膜隨月經週期增厚與剝落
- 子宮壁含平滑肌，分娩時收縮

### 陰道（vagina）
- 為產道與月經排出的通道
- 環境呈**弱酸性**，具防禦功能

## 月經週期（menstrual cycle）

### 概述
- 週期約 **28 天**（因人而異）
- 涉及**卵巢週期**與**子宮內膜週期**的同步變化

### 卵巢週期
#### 濾泡期（follicular phase）— 第 1～14 天
- 數個濾泡開始發育，通常僅**一個**發育成熟為**葛拉夫氏濾泡（Graafian follicle）**
- 濾泡分泌**雌激素（estrogen）**，濃度逐漸升高
- 雌激素促進子宮內膜增生

#### 排卵（ovulation）— 約第 14 天
- 雌激素濃度達高峰 → 正回饋 → 引發腦下腺前葉大量釋放 **LH（黃體成長激素）**
- **LH 激增（LH surge）**為排卵的直接觸發因素
- 成熟濾泡破裂，卵子排入腹腔，被輸卵管繖部承接

#### 黃體期（luteal phase）— 第 15～28 天
- 排卵後的濾泡轉變為**黃體（corpus luteum）**
- 黃體分泌**黃體素（progesterone）**與少量雌激素
- 黃體素促進子宮內膜**持續增厚**與**腺體分泌**，為胚胎著床做準備
- 若無受精 → 黃體退化為**白體** → 黃體素與雌激素急降 → 子宮內膜剝落 → **月經來潮**

### 子宮內膜週期
- **月經期（menstruation）**：第 1～5 天，子宮內膜剝落出血
- **增生期（proliferative phase）**：第 6～14 天，雌激素刺激內膜增生
- **分泌期（secretory phase）**：第 15～28 天，黃體素刺激內膜腺體分泌

## 激素調控

### 下視丘-腦下腺-性腺軸（HPG axis）
- **下視丘**分泌 **GnRH（促性腺激素釋放激素）**
- GnRH 刺激**腦下腺前葉**分泌 FSH 與 LH

### FSH（濾泡刺激素）
- 促進卵巢**濾泡發育**
- 促進睪丸**精子生成**（刺激乙狀細胞）

### LH（黃體成長激素）
- 女性：觸發**排卵**、促進**黃體形成**與黃體素分泌
- 男性：刺激**萊氏細胞**分泌睪固酮

### 雌激素（estrogen）
- 主要由**卵巢濾泡**分泌
- 促進女性**第二性徵**發育：乳房發育、骨盆變寬、皮下脂肪分布
- 促進子宮內膜增生
- **低濃度**對腦下腺呈**負回饋**（抑制 FSH、LH）
- **高濃度**（排卵前）對腦下腺呈**正回饋**（引發 LH 激增）

### 黃體素（progesterone）
- 主要由**黃體**分泌
- 維持子宮內膜穩定，利於胚胎著床
- **抑制**子宮收縮
- 對腦下腺呈**負回饋**，抑制 FSH 與 LH 分泌
- 使基礎體溫**微升**（約 0.3～0.5°C）

## 受精與著床

### 受精（fertilization）
- 正常發生於**輸卵管壺腹部**
- 精子頂體釋放酵素，穿透卵子外層的**放射冠**與**透明帶（zona pellucida）**
- 精子與卵子細胞膜融合 → 觸發**皮質反應（cortical reaction）**
- 皮質反應使透明帶硬化 → **阻止多精入卵**
- 精卵細胞核融合形成**受精卵（zygote）**（2n）

### 著床（implantation）
- 受精卵在輸卵管中進行**卵裂（cleavage）**
- 受精後約第 3 天形成**桑椹胚（morula）**
- 約第 4～5 天形成**囊胚（blastocyst）**
- 受精後約**第 6～7 天**，囊胚**著床**於子宮內膜

## 胚胎發育

### 早期發育
- 囊胚分為**內細胞團（inner cell mass）**與**滋養層（trophoblast）**
- 內細胞團發育為**胚胎本體**
- 滋養層參與形成**胎盤（placenta）**

### 胎盤功能
- 為母體與胎兒之間的**物質交換**場所
- 母血與胎血**不直接混合**
- 交換物質：O₂、CO₂、營養素、廢物
- 分泌**人類絨毛膜促性腺激素（hCG）**：維持黃體功能，為**驗孕**的偵測指標
- 後期分泌雌激素與黃體素，取代黃體功能

### 三胚層分化
- **外胚層（ectoderm）**：發育為皮膚表皮、神經系統、感覺器官
- **中胚層（mesoderm）**：發育為肌肉、骨骼、循環系統、泌尿系統、生殖系統
- **內胚層（endoderm）**：發育為消化道內壁、呼吸道內壁、肝臟、胰臟

### 懷孕期程
- 人類懷孕期約 **38～40 週（約 280 天）**
- **第一孕期（1～12 週）**：器官形成期，胚胎對致畸因子最敏感
- **第二孕期（13～24 週）**：器官持續發育，胎兒快速成長
- **第三孕期（25～40 週）**：器官成熟，胎兒體重明顯增加
