# 相對論簡介

## 狹義相對論的基本假設
### 第一假設（相對性原理）
- 物理定律在所有慣性座標系中形式相同
- 沒有任何實驗可以區分靜止與等速運動
- 推廣伽利略的力學相對性至所有物理定律
### 第二假設（光速恆定原理）
- 真空中光速 c = 3 × 10⁸ m/s，對所有觀察者相同
- 光速與光源或觀察者的運動狀態無關
- 邁克生-莫雷實驗（Michelson-Morley）: 未偵測到以太，支持光速恆定

## 時間膨脹
### 時間膨脹效應
- 運動的時鐘變慢（相對於靜止觀察者）
- 公式: Δt = γΔt₀ = Δt₀ / √(1 - v²/c²)
  - Δt₀: 固有時間（proper time，靜止座標系中的時間）
  - Δt: 運動座標系中觀察到的時間
  - γ: 勞侖茲因子 = 1/√(1 - v²/c²) ≥ 1
### 實驗證據
- μ介子壽命延長: 宇宙射線產生的 μ 子因高速運動壽命延長
- 原子鐘實驗: 搭飛機的原子鐘比地面的慢
- GPS 衛星: 需校正相對論時間差

## 長度收縮
### 長度收縮效應
- 運動方向上的物體長度縮短
- 公式: L = L₀ / γ = L₀√(1 - v²/c²)
  - L₀: 固有長度（proper length，靜止座標系中的長度）
  - L: 運動中觀察到的長度
- 僅在運動方向上收縮，垂直方向不變
### 注意事項
- 長度收縮與時間膨脹是同一現象的不同面向
- v → 0 時: γ → 1，回復到日常經驗
- v → c 時: γ → ∞，時間無限膨脹、長度趨近於零

## 質能等價
- E = mc²: 質量與能量等價
- 靜止能量: E₀ = m₀c²
- 總能量: E = γm₀c²
- 動能: KE = (γ - 1)m₀c²
- 低速近似: KE ≈ ½m₀v²（回到古典力學）

## 相對論動量
- 相對論動量: p = γmv
- 速度越接近光速，動量增加越快
- 無法將有質量物體加速到光速（需無限大能量）
- 能量-動量關係: E² = (pc)² + (m₀c²)²
