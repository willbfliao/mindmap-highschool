# 細胞的能量來源

## ATP — 細胞的能量貨幣

### ATP 的結構
- **ATP（adenosine triphosphate，三磷酸腺苷）**
- 由腺嘌呤（adenine）+ 核糖（ribose）+ 三個磷酸基組成
- 末端磷酸鍵為**高能磷酸鍵**

### ATP 的功能
- 水解時釋放能量：ATP → ADP + Pᵢ + 能量
- 為細胞各種活動提供直接能量來源
- 能量用於：肌肉收縮、主動運輸、生物合成、神經傳導等
- ATP 可循環再生：ADP + Pᵢ + 能量 → ATP

## 光合作用基礎（photosynthesis）

### 概述
- 發生於**葉綠體**中
- 總反應式：**6CO₂ + 6H₂O →（光能）→ C₆H₁₂O₆ + 6O₂**
- 將**光能**轉化為**化學能**，儲存於葡萄糖中
- 為地球上大多數生態系的**能量基礎**

### 光合作用的條件
- **光線**：提供能量
- **二氧化碳（CO₂）**：碳的來源
- **水（H₂O）**：氫的來源，分解後釋放 O₂
- **葉綠素（chlorophyll）**：吸收光能的色素

### 光合作用的兩個階段
#### 光反應（light reactions）
- 發生於**類囊體膜（thylakoid membrane）**
- 需要**光線**
- 水的光解：H₂O → O₂ + H⁺ + e⁻
- 產生 **ATP** 與 **NADPH**（供暗反應使用）
- O₂ 為**副產物**，由水分解而來

#### 暗反應 / 碳固定（Calvin cycle）
- 發生於**基質（stroma）**
- 不直接需要光，但需要光反應的產物
- CO₂ 被固定為有機碳：CO₂ → C₃ → G3P → 葡萄糖
- 消耗光反應產生的 ATP 與 NADPH

### 影響光合作用速率的因素
- **光照強度**：光補償點以上光合速率隨光照增加而增加，至**光飽和點**後不再增加
- **CO₂ 濃度**：適度增加可提高光合速率
- **溫度**：酵素催化反應受溫度影響，過高或過低均降低速率

## 呼吸作用基礎（cellular respiration）

### 概述
- 發生於**細胞質**與**粒線體**
- 總反應式：**C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O + 能量（ATP）**
- 分解有機物，釋放儲存的化學能
- 所有生物（包括植物）都進行細胞呼吸

### 有氧呼吸的三個階段
#### 糖解作用（glycolysis）
- 發生於**細胞質**
- 葡萄糖（6C）→ 2 分子丙酮酸（3C）
- 淨產生 **2 ATP** 和 **2 NADH**
- **不需要氧氣**

#### 克氏循環（Krebs cycle / citric acid cycle）
- 發生於**粒線體基質**
- 丙酮酸 → 乙醯 CoA → 進入循環
- 產生 CO₂、NADH、FADH₂ 與少量 ATP

#### 電子傳遞鏈（electron transport chain）
- 發生於**粒線體內膜（嵴）**
- NADH 與 FADH₂ 將電子傳遞給一系列載體
- 最終電子接受者為 **O₂**，生成 H₂O
- 經由**氧化磷酸化**產生大量 **ATP**（約 34 個）

### 有氧呼吸的總產量
- 1 分子葡萄糖完全氧化約產生 **36～38 個 ATP**
- 大部分 ATP 來自電子傳遞鏈

### 無氧呼吸 / 發酵（anaerobic respiration / fermentation）
- 在**缺氧**條件下進行
- 只經過糖解作用，淨產生 **2 ATP**（效率遠低於有氧呼吸）
- **酒精發酵**：丙酮酸 → 乙醇 + CO₂（酵母菌）
- **乳酸發酵**：丙酮酸 → 乳酸（動物肌肉劇烈運動時、乳酸菌）

## 光合作用與呼吸作用的比較

### 比較表
| 項目 | 光合作用 | 呼吸作用 |
|------|----------|----------|
| 發生場所 | 葉綠體 | 粒線體（及細胞質） |
| 能量轉換 | 光能 → 化學能 | 化學能 → ATP |
| 反應物 | CO₂ + H₂O | C₆H₁₂O₆ + O₂ |
| 產物 | C₆H₁₂O₆ + O₂ | CO₂ + H₂O + ATP |
| 進行條件 | 需光 | 隨時進行 |
| 進行生物 | 自營生物 | 所有生物 |

### 兩者的關係
- 光合作用的產物（葡萄糖、O₂）為呼吸作用的反應物
- 呼吸作用的產物（CO₂、H₂O）為光合作用的反應物
- 形成**物質與能量的循環**
- 植物白天光合作用 > 呼吸作用；夜晚僅進行呼吸作用
