# 碰撞

## 碰撞的分類
### 依動能是否守恆
- 彈性碰撞（elastic collision）: 動量守恆 + 動能守恆
- 非彈性碰撞（inelastic collision）: 動量守恆，動能不守恆（部分轉為熱能、聲能）
- 完全非彈性碰撞（perfectly inelastic）: 碰後黏在一起，動能損失最大
### 依碰撞方向
- 正碰（head-on）: 碰撞沿連心線方向
- 斜碰（oblique）: 碰撞不沿連心線方向

## 一維彈性碰撞
### 基本公式
- 動量守恆: m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'
- 動能守恆: ½m₁v₁² + ½m₂v₂² = ½m₁v₁'² + ½m₂v₂'²
### 碰後速度公式
- v₁' = [(m₁-m₂)v₁ + 2m₂v₂] / (m₁+m₂)
- v₂' = [(m₂-m₁)v₂ + 2m₁v₁] / (m₁+m₂)
### 特殊情況
- 等質量碰撞（m₁ = m₂）: 速度互換，v₁' = v₂、v₂' = v₁
- 輕球撞重牆（m₁ ≪ m₂）: 輕球近乎原速反彈
- 重球撞輕球（m₁ ≫ m₂）: 輕球以約 2v₁ 飛出

## 一維完全非彈性碰撞
- 碰後合體: m₁v₁ + m₂v₂ = (m₁+m₂)v'
- v' = (m₁v₁ + m₂v₂) / (m₁+m₂)
- 動能損失: ΔKE = ½ · m₁m₂/(m₁+m₂) · (v₁-v₂)²
- 應用: 子彈射入木塊、車輛碰撞

## 二維碰撞
### 分析方法
- x 方向動量守恆: m₁v₁ₓ + m₂v₂ₓ = m₁v₁ₓ' + m₂v₂ₓ'
- y 方向動量守恆: m₁v₁ᵧ + m₂v₂ᵧ = m₁v₁ᵧ' + m₂v₂ᵧ'
- 若為彈性碰撞再加動能守恆方程
### 二維等質量彈性碰撞
- 碰後兩物體速度方向夾 90°（靶球靜止時）
- 撞球（pool）的物理原理

## 恢復係數
- 恢復係數（coefficient of restitution）: e = |v₂' - v₁'| / |v₁ - v₂|
- 彈性碰撞: e = 1
- 完全非彈性碰撞: e = 0
- 一般碰撞: 0 < e < 1
