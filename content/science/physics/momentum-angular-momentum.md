# 動量與角動量

## 動量
### 動量的定義
- 動量（momentum）: p = mv（向量）
- 向量: 方向與速度相同
- 單位: kg·m/s
- 動量越大，越難改變運動狀態
### 動量定理（衝量-動量定理）
- 衝量（impulse）: J = FΔt = Δp（向量）
- 合力的衝量 = 動量的變化量
- F-t 圖下的面積 = 衝量
- 應用: 安全氣囊延長碰撞時間以減小衝擊力

## 動量守恆定律
### 守恆條件
- 系統所受合外力為零（或內力遠大於外力）
- 動量守恆: Σp初 = Σp末
- 即 m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'
### 二維動量守恆
- x 方向動量守恆: Σpₓ_初 = Σpₓ_末
- y 方向動量守恆: Σpᵧ_初 = Σpᵧ_末
- 分量獨立守恆

## 質心運動
- 質心速度: v_cm = Σmᵢvᵢ / Σmᵢ
- 質心動量: P_cm = Mv_cm = Σmᵢvᵢ
- 合外力為零時質心速度不變
- 煙火爆炸: 質心仍沿拋物線運動

## 角動量
### 角動量的定義
- 角動量（angular momentum）: L = mvr sinθ
- 質點繞固定軸: L = mr²ω = Iω
- 轉動慣量（moment of inertia）: I = Σmᵢrᵢ²
- 單位: kg·m²/s
### 角動量守恆
- 條件: 系統所受合外力矩為零
- L_初 = L_末，即 Iω = 常數
- 花式溜冰: 縮手 → r 減小 → ω 增大 → 轉速加快
- 克卜勒第二定律: 行星角動量守恆的結果
