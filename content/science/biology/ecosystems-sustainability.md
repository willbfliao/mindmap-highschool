# 生態系與環境

## 生態系的組成

### 生態系（ecosystem）的定義
- 特定區域內所有**生物**（生物群集）與其**非生物環境**的交互作用系統
- 生物組成：生產者、消費者、分解者
- 非生物環境：陽光、水、溫度、土壤、空氣

### 生產者（producers / autotrophs）
- **自營生物**，能將無機物合成有機物
- 主要為行**光合作用**的生物：植物、藻類、藍綠菌
- 少數為**化學自營**：硝化細菌、硫化細菌（深海熱泉生態系）
- 為生態系的**能量基礎**

### 消費者（consumers / heterotrophs）
- **初級消費者（primary consumer）**：草食性動物（如兔、牛、毛毛蟲）
- **次級消費者（secondary consumer）**：以初級消費者為食（如蛙、小型肉食獸）
- **三級消費者（tertiary consumer）**：頂級掠食者（如老鷹、獅子）
- **雜食者（omnivore）**：取食多個營養階層（如人、熊）

### 分解者（decomposers）
- 分解**死亡有機物**，將有機物轉化為無機物
- 主要為**細菌**與**真菌**
- 使養分回歸環境，維持**物質循環**
- 若無分解者 → 養分被鎖在死亡有機體中 → 生態系崩潰

## 能量流動（energy flow）

### 食物鏈（food chain）
- 能量從生產者經消費者逐級傳遞的**線性途徑**
- 例：草 → 兔 → 狐 → 鷹
- 每一級稱為一個**營養階層（trophic level）**

### 食物網（food web）
- 多條食物鏈**交織**而成的複雜網絡
- 更能反映生態系中真實的能量傳遞關係
- 食物網越複雜 → 生態系越**穩定**

### 能量傳遞效率
- 每個營養階層只有約 **10%** 的能量傳遞到下一階層
- 其餘能量以**熱能**形式散失（呼吸作用）
- **10% 法則（Lindeman's 10% rule）**
- 因此食物鏈通常不超過 **4～5 個**營養階層
- **生態金字塔**：
  - **能量金字塔**：永遠為正金字塔（能量逐級遞減）
  - **生物量金字塔**：通常為正金字塔
  - **數量金字塔**：有時可倒轉（如一棵大樹上許多昆蟲）

### 能量流動的特性
- 能量流動為**單向**、**不可循環**
- 太陽能 → 生產者固定 → 沿食物鏈傳遞 → 最終以熱能散失
- 需要持續的**太陽能輸入**維持生態系運作

## 物質循環（biogeochemical cycles）

### 碳循環（carbon cycle）
- **光合作用**：大氣 CO₂ → 有機碳（植物體）
- **呼吸作用**：有機碳 → CO₂ 釋回大氣
- **分解作用**：死亡生物的有機碳 → CO₂
- **化石燃料燃燒**：煤、石油、天然氣 → 大量 CO₂ 釋入大氣
- **海洋**：吸收大量 CO₂（碳匯）
- 人類活動大幅增加大氣 CO₂ → **溫室效應加劇**

### 氮循環（nitrogen cycle）
- 大氣 N₂ 佔空氣 **78%**，但大多數生物**無法直接利用**
- **固氮作用（nitrogen fixation）**：
  - **生物固氮**：根瘤菌、固氮藍綠菌將 N₂ → NH₃/NH₄⁺
  - **閃電固氮**：高能使 N₂ + O₂ → NO → NO₃⁻
  - **工業固氮**：哈柏法（Haber process）合成氨
- **硝化作用（nitrification）**：
  - NH₄⁺ → NO₂⁻ → **NO₃⁻**（由硝化細菌進行）
  - 植物主要吸收 **NO₃⁻** 與 **NH₄⁺**
- **同化作用**：植物將 NO₃⁻ 合成胺基酸、蛋白質
- **氨化作用（ammonification）**：分解者將有機氮 → NH₄⁺
- **脫氮作用（denitrification）**：脫氮菌將 NO₃⁻ → N₂ 釋回大氣

### 水循環（water cycle）
- **蒸發**：水面、土壤、植物蒸散 → 水蒸氣
- **凝結**：水蒸氣 → 雲
- **降水**：雨、雪回到地表
- **地表逕流**與**地下水滲透**
- 植物的蒸散作用是水循環的重要環節

### 磷循環（phosphorus cycle）
- **無氣態階段**（與碳、氮循環不同）
- 磷主要存在於**岩石與沉積物**中
- 岩石風化 → 磷酸鹽溶入水中 → 被植物吸收
- 經食物鏈傳遞 → 分解後回歸土壤
- 磷為**限制性養分**：常為生態系生產力的限制因子
- 過量磷流入水體 → **優養化**

## 生物群系（biomes）

### 陸域生物群系
- 由**氣溫**與**降雨量**決定
- **熱帶雨林**：高溫多雨，物種多樣性最高
- **熱帶草原**：明顯乾濕季，大型草食動物群
- **沙漠**：極度乾燥，生物具有保水適應
- **溫帶草原**：季節溫差大，肥沃黑土
- **溫帶落葉林**：四季分明，秋季落葉
- **針葉林（北方森林 taiga）**：寒冷，常綠針葉樹
- **凍原（tundra）**：極寒，永凍土，矮小植被

### 水域生態系
- **淡水生態系**：河流、湖泊、溼地
- **海洋生態系**：
  - **潮間帶**：受潮汐影響
  - **近海區**：大陸棚，生產力高
  - **遠洋區**：大洋深處
  - **珊瑚礁**：「海洋的熱帶雨林」，生物多樣性極高
  - **深海熱泉**：以化學自營細菌為基礎

## 人類與環境的永續發展

### 全球環境議題

#### 溫室效應與全球暖化
- **溫室氣體**：CO₂、CH₄、N₂O、氟氯碳化物
- 大量燃燒化石燃料 → CO₂ 濃度升高 → **增強溫室效應**
- 影響：全球平均溫度上升、冰川融化、海平面上升、極端氣候增加
- **巴黎協定（Paris Agreement）**：控制全球增溫在 1.5～2°C 以內

#### 臭氧層破壞
- **臭氧層（ozone layer）**：位於平流層，吸收紫外線
- **氟氯碳化物（CFCs）**破壞臭氧分子
- 南極上空出現**臭氧洞**
- **蒙特婁議定書**：禁用 CFCs

#### 酸雨（acid rain）
- SO₂、NOₓ 排放 → 與水結合形成硫酸、硝酸
- pH < 5.0 的降水
- 損害森林、湖泊酸化、建築腐蝕

### 永續發展（sustainable development）
- **滿足當代需求，而不損害後代滿足其需求的能力**
- 三大面向：**環境保護、經濟發展、社會公平**

#### 永續策略
- **減少碳排放**：發展再生能源（太陽能、風能）
- **循環經濟**：減量、重複使用、回收
- **永續農業**：有機農法、生物防治
- **保育生物多樣性**：設立保護區、復育瀕危物種
- **環境教育**：提升公民環境素養

### 生態足跡（ecological footprint）
- 維持一個人生活所需的**生物生產性面積**
- 用來衡量人類對地球資源的消耗程度
- 目前全人類的生態足跡已**超過地球的承載力**（生態赤字）
- 減少個人生態足跡：節能減碳、減少浪費、選擇在地食材
