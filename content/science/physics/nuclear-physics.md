# 原子核物理

## 原子核結構
### 核子
- 質子（proton）: 帶正電，質量 ≈ 1.673 × 10⁻²⁷ kg
- 中子（neutron）: 不帶電，質量略大於質子
- 核子（nucleon）: 質子與中子的統稱
### 原子序與質量數
- 原子序（Z）: 質子數，決定元素種類
- 質量數（A）: 質子數 + 中子數（A = Z + N）
- 同位素（isotope）: 質子數相同但中子數不同
  - 例: ¹H、²H（氘）、³H（氚）
### 核力
- 強核力: 束縛核子的力，作用距離極短（≈ 10⁻¹⁵ m）
- 不分質子或中子，同等吸引
- 短距離內遠大於庫侖排斥力
### 結合能
- 質量虧損（mass defect）: Δm = Zm_p + Nm_n - M_nucleus
- 結合能: E_b = Δmc²
- 平均結合能: E_b/A（每核子平均結合能）
- 鐵-56（⁵⁶Fe）平均結合能最大 → 最穩定

## 放射性衰變
### α 衰變
- α 粒子: ⁴He 原子核（2 質子 + 2 中子）
- ᴬ_Z X → ᴬ⁻⁴_(Z-2) Y + ⁴₂He
- 質量數減 4，原子序減 2
- 穿透力最弱（紙張即可擋住）
### β 衰變
- β⁻ 衰變: 中子 → 質子 + 電子 + 反微中子
  - ᴬ_Z X → ᴬ_(Z+1) Y + e⁻ + 反微中子
  - 原子序加 1，質量數不變
- β⁺ 衰變: 質子 → 中子 + 正子 + 微中子
  - ᴬ_Z X → ᴬ_(Z-1) Y + e⁺ + ν
### γ 衰變
- γ 射線: 高能量電磁波（光子）
- 原子核從激發態躍遷至低能態時放出
- 原子序與質量數不變
- 穿透力最強（需厚鉛板或混凝土屏蔽）
### 半衰期
- 半衰期（half-life, t₁/₂）: 放射性核種衰變一半所需時間
- N(t) = N₀ × (½)^(t/t₁/₂)
- 與外在條件（溫度、壓力）無關
- 碳-14 定年法: t₁/₂ = 5730 年，測定考古樣本年代

## 核分裂
### 核分裂反應
- 核分裂（nuclear fission）: 重原子核分裂為較輕的原子核
- ²³⁵U + n → ¹⁴¹Ba + ⁹²Kr + 3n（典型反應之一）
- 釋放大量能量（質量虧損轉為能量）
### 連鎖反應
- 每次分裂釋放 2~3 個中子 → 引發更多分裂
- 臨界質量（critical mass）: 維持連鎖反應的最少燃料量
- 控制棒: 吸收多餘中子以控制反應速率
### 核能發電
- 核反應爐: 控制連鎖反應的速率
- 冷卻劑: 帶走熱量產生蒸汽推動渦輪
- 優點: 不排放 CO₂、能量密度高
- 缺點: 核廢料處理、輻射安全風險

## 核融合
### 核融合反應
- 核融合（nuclear fusion）: 輕原子核結合為較重的原子核
- 太陽能量來源: 4¹H → ⁴He + 2e⁺ + 2ν + 能量（質子-質子鏈）
- 條件: 需極高溫度（> 10⁷ K）以克服庫侖排斥
### 核融合與核分裂的比較
- 核融合: 每核子釋放能量更多、燃料豐富（氫）、產物無放射性
- 目前挑戰: 電漿約束困難，尚無法穩定商業發電
- 氫彈: 不可控的核融合（以核分裂引爆）
