# 電流與電路

## 電流與電阻
### 電流
- 電流（current）: I = ΔQ/Δt（單位時間通過截面的電荷量）
- 單位: 安培（A）= C/s
- 電流方向: 正電荷移動方向（與電子移動方向相反）
- 電流密度: J = I/A = nev_d（n: 載子濃度，v_d: 漂移速度）
### 電阻
- 電阻（resistance）: R = ρL/A
  - ρ: 電阻率，L: 導線長度，A: 截面積
- 單位: 歐姆（Ω）= V/A
- 溫度效應: 金屬電阻隨溫度升高而增大

## 歐姆定律
### 定律內容
- 歐姆定律（Ohm's law）: V = IR
- 適用於歐姆材料（金屬導體等）
- I-V 圖為通過原點的直線
### 功率
- 電功率: P = IV = I²R = V²/R
- 焦耳熱: Q = I²Rt（電能轉換為熱能）
- 單位: 瓦特（W）

## 克希荷夫定律
### 克希荷夫電流定律（KCL）
- 節點定律: 流入節點的電流總和 = 流出節點的電流總和
- ΣI_in = ΣI_out
- 本質: 電荷守恆
### 克希荷夫電壓定律（KVL）
- 迴路定律: 沿任一封閉迴路，電壓升降代數和為零
- ΣV = 0（繞一圈回到起點）
- 本質: 能量守恆
### 電路分析
- 串聯電阻: R_total = R₁ + R₂ + R₃ + ...
  - 電流處處相同，電壓分配
- 並聯電阻: 1/R_total = 1/R₁ + 1/R₂ + 1/R₃ + ...
  - 電壓處處相同，電流分配

## 電容與 RC 電路
### 電容器
- 電容（capacitance）: C = Q/V
- 單位: 法拉（F）= C/V
- 平行板電容: C = ε₀A/d（A: 面積，d: 間距）
- 介電質: C = κε₀A/d（κ: 介電常數 > 1）
### 電容器的能量
- 儲能: U = ½CV² = ½QV = Q²/(2C)
### 電容的串並聯
- 串聯: 1/C_total = 1/C₁ + 1/C₂ + ...（電荷相同）
- 並聯: C_total = C₁ + C₂ + ...（電壓相同）
### RC 電路
- 充電: V_C(t) = V₀(1 - e^(-t/RC))
- 放電: V_C(t) = V₀ · e^(-t/RC)
- 時間常數: τ = RC（充電至 63.2% 或放電至 36.8%）
- 5τ 後近似完全充電或放電
