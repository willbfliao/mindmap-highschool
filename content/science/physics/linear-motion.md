# 直線運動

## 位移與路徑長
- 位移（displacement）: 起點到終點的直線距離，向量
- 路徑長（distance）: 實際行走路線長度，純量
- 位移大小 $\leq$ 路徑長

## 速度與速率
### 平均速度與瞬時速度
- 平均速度（average velocity）: $v_{avg} = \frac{\Delta x}{\Delta t}$（向量）
- 平均速率（average speed）: $\frac{\text{路徑長}}{\text{時間}}$（純量）
- 瞬時速度（instantaneous velocity）: $\Delta t \to 0$ 時的平均速度
### x-t 圖與 v-t 圖
- x-t 圖斜率 = 速度
- v-t 圖斜率 = 加速度
- v-t 圖下面積 = 位移

## 加速度
- 加速度（acceleration）: $a = \frac{\Delta v}{\Delta t}$
- 加速度方向與速度**同向** → 加速（速率增加）
- 加速度方向與速度**反向** → 減速（速率減少）
- 正負號僅代表方向（相對於所定義的正方向），與加/減速無必然對應
- 單位: $\text{m/s}^2$

## 等加速度運動
### 三大公式
- $v = v_0 + at$: 速度-時間關係
- $x = v_0 t + \frac{1}{2}at^2$: 位移-時間關係
- $v^2 = v_0^2 + 2ax$: 速度-位移關係
### 補充公式
- 平均速度: $v_{avg} = \frac{v_0 + v}{2}$（僅適用等加速度）
- 第 n 秒位移: $x_n = v_0 + a(n - \frac{1}{2})$

## 自由落體
- 定義: 僅受重力作用、初速為零的運動
- 加速度: $a = g \approx 9.8 \text{ m/s}^2$（向下為正）
- 公式: $v = gt$、$h = \frac{1}{2}gt^2$、$v^2 = 2gh$
- 與質量無關: 真空中羽毛與鐵球同時落地
- 伽利略落體實驗: 以斜面實驗推翻亞里斯多德重物先落地說
  - 比薩斜塔實驗為未經證實的傳說

## 鉛直上拋
- 初速 $v_0$ 向上，加速度 $g$ 向下
- 最高點: $v = 0$，上升時間 $t = \frac{v_0}{g}$
- 最大高度: $h = \frac{v_0^2}{2g}$
- 上升與下降時間相同（忽略空氣阻力）
- 回到拋出點時速率等於初速率
