# 電流與電路

## 電流與電阻
### 電流
- 電流（current）: $I = \frac{\Delta Q}{\Delta t}$（單位時間通過截面的電荷量）
- 單位: 安培（A）= $\text{C/s}$
- 電流方向: 正電荷移動方向（與電子移動方向相反）
- 電流密度: $J = \frac{I}{A} = nev_{d}$（$n$: 載子濃度，$v_{d}$: 漂移速度）
### 電阻
- 電阻（resistance）: $R = \frac{\rho L}{A}$
  - $\rho$: 電阻率，$L$: 導線長度，$A$: 截面積
- 單位: 歐姆（$\Omega$）= $\text{V/A}$
- 溫度效應: 金屬電阻隨溫度升高而增大

## 歐姆定律
### 定律內容
- 歐姆定律（Ohm's law）: $V = IR$
- 適用於歐姆材料（金屬導體等）
- $I$-$V$ 圖為通過原點的直線
### 功率
- 電功率: $P = IV = I^{2}R = \frac{V^{2}}{R}$
- 焦耳熱: $Q = I^{2}Rt$（電能轉換為熱能）
- 單位: 瓦特（W）

## 克希荷夫定律
### 克希荷夫電流定律（KCL）
- 節點定律: 流入節點的電流總和 = 流出節點的電流總和
- $\sum I_{in} = \sum I_{out}$
- 本質: 電荷守恆
### 克希荷夫電壓定律（KVL）
- 迴路定律: 沿任一封閉迴路，電壓升降代數和為零
- $\sum V = 0$（繞一圈回到起點）
- 本質: 能量守恆
### 電路分析
- 串聯電阻: $R_{total} = R_{1} + R_{2} + R_{3} + \cdots$
  - 電流處處相同，電壓分配
- 並聯電阻: $\frac{1}{R_{total}} = \frac{1}{R_{1}} + \frac{1}{R_{2}} + \frac{1}{R_{3}} + \cdots$
  - 電壓處處相同，電流分配

## 電容與 RC 電路
### 電容器
- 電容（capacitance）: $C = \frac{Q}{V}$
- 單位: 法拉（F）= $\text{C/V}$
- 平行板電容: $C = \frac{\varepsilon_{0} A}{d}$（$A$: 面積，$d$: 間距）
- 介電質: $C = \frac{\kappa \varepsilon_{0} A}{d}$（$\kappa$: 介電常數 $> 1$）
### 電容器的能量
- 儲能: $U = \frac{1}{2}CV^{2} = \frac{1}{2}QV = \frac{Q^{2}}{2C}$
### 電容的串並聯
- 串聯: $\frac{1}{C_{total}} = \frac{1}{C_{1}} + \frac{1}{C_{2}} + \cdots$（電荷相同）
- 並聯: $C_{total} = C_{1} + C_{2} + \cdots$（電壓相同）
### RC 電路
- 充電: $V_{C}(t) = V_{0}(1 - e^{-t/RC})$
- 放電: $V_{C}(t) = V_{0} \cdot e^{-t/RC}$
- 時間常數: $\tau = RC$（充電至 63.2% 或放電至 36.8%）
- $5\tau$ 後近似完全充電或放電
