# 進階遺傳學

## 非孟德爾遺傳

### 不完全顯性（incomplete dominance）
- 異型合子的表現型為兩純系親代的**中間型**
- 例：紫茉莉花色 — RR（紅花）× WW（白花）→ RW（**粉紅花**）
- F₂ 表現型比 = **1 紅：2 粉紅：1 白**

### 共顯性（codominance）
- 異型合子**同時表現**兩種等位基因的性狀
- 例：人類 ABO 血型 — Iᴬ 與 Iᴮ 為共顯性
  - IᴬIᴮ → **AB 型**（紅血球同時表現 A 抗原與 B 抗原）

### 複等位基因（multiple alleles）
- 一個基因座存在**三種或更多**等位基因（但個體只有兩個）
- 例：ABO 血型由 **Iᴬ、Iᴮ、i** 三個等位基因控制
  - Iᴬ 與 Iᴮ 對 i 為顯性
  - 基因型與表現型：IᴬIᴬ 或 Iᴬi → A 型；IᴮIᴮ 或 Iᴮi → B 型；IᴬIᴮ → AB 型；ii → O 型
- 💡 聯想：ABO 血型就像班級投票—A 和 B 都是強勢候選人（對 i 顯性），兩個同時選上就是聯合執政（AB 型共顯性）

### 上位作用（epistasis）
- 一個基因的表現**遮蓋或影響**另一個非等位基因的表現
- 例：拉布拉多犬毛色
  - B 基因控制色素（B = 黑色，b = 棕色）
  - E 基因控制色素沉積（ee → 不沉積 → 黃色，不論 B 基因）
  - 雙雜交 BbEe × BbEe → 9 黑：3 棕：4 黃（修正的 9:3:3:1）

### 多基因遺傳（polygenic inheritance）
- 一個性狀由**多個基因**共同控制
- 表現型呈**連續變異（continuous variation）**，呈常態分布
- 受**環境**影響大
- 例：膚色、身高、體重

### 基因多效性（pleiotropy）
- 一個基因影響**多個**性狀
- 例：鐮刀形紅血球貧血症的 HbS 基因
  - 影響紅血球形狀、血液循環、器官功能等多種表現

## 連鎖與互換（linkage & crossing over）

### 基因連鎖（linkage）
- 位於**同一條染色體**上的基因傾向一起遺傳
- 不符合孟德爾的自由配合律
- **摩根（Morgan）**發現果蠅基因連鎖現象
- 完全連鎖：同一染色體上的基因永遠一起遺傳

### 互換與重組（recombination）
- 減數分裂前期 I，同源染色體的非姊妹染色分體發生**互換**
- 互換打破連鎖，產生**重組型配子**
- **重組頻率（recombination frequency）**：
  - 重組頻率 = 重組型後代數 / 總後代數 × 100%
  - 兩基因距離越遠 → 互換機率越大 → 重組頻率越高
  - 最大重組頻率為 **50%**（等同獨立分配）

### 基因圖譜（genetic mapping）
- 利用重組頻率推算基因在染色體上的**相對距離**
- **1 map unit（centiMorgan, cM）= 1% 重組頻率**
- 三點試交法可確定基因在染色體上的排列順序

## 性聯遺傳進階

### X 染色體性聯遺傳
- 男性只有一個 X 染色體 → 性聯隱性等位基因**直接表現**（半合子 hemizygous）
- 女性需要兩個隱性等位基因才會表現
- 典型遺傳模式：
  - 攜帶者母親（XᴮXᵇ）× 正常父親（XᴮY）→ 兒子有 1/2 機率患病
  - **交叉遺傳（criss-cross inheritance）**：父親的 X 染色體傳給女兒
- 例：色盲、血友病、杜乓氏肌肉萎縮症

### 性聯顯性遺傳
- 位於 X 染色體上的**顯性**基因
- 患病父親 → **所有女兒**患病，所有兒子正常
- 例：抗維生素 D 性佝僂症

### X 染色體去活化（X-inactivation）
- 雌性哺乳動物的兩條 X 染色體中，一條在胚胎早期**隨機去活化**
- 去活化的 X 形成**乓爾體（Barr body）**
- 導致雌性為**鑲嵌體（mosaic）**
- 例：三色貓（calico cat）的毛色斑紋

## 族群遺傳學（population genetics）

### 基因庫（gene pool）
- 一個族群中所有個體的**全部等位基因的總和**
- **等位基因頻率（allele frequency）**：特定等位基因在族群中的比例
- **基因型頻率（genotype frequency）**：特定基因型在族群中的比例

### 哈地-溫伯格平衡（Hardy-Weinberg equilibrium）
- 在**理想族群**中，等位基因頻率與基因型頻率**世代不變**
- **前提條件**（五個假設）：
  1. 族群**非常大**（無遺傳漂變）
  2. **無突變**
  3. **無遷移**（無基因流動）
  4. **隨機交配**
  5. **無天擇**（所有基因型的適應度相等）
- **公式**：
  - 等位基因頻率：**p + q = 1**（p = 顯性等位基因頻率，q = 隱性等位基因頻率）
  - 基因型頻率：**p² + 2pq + q² = 1**
  - p² = AA，2pq = Aa，q² = aa

### 改變等位基因頻率的因素
- **天擇（natural selection）**：有利基因型的頻率增加
- **基因突變（mutation）**：產生新的等位基因
- **遺傳漂變（genetic drift）**：小族群中等位基因頻率的隨機波動
  - **瓶頸效應（bottleneck effect）**：災難後存活個體基因組成偏離原族群
  - **建立者效應（founder effect）**：少數移民建立新族群
- **基因流動（gene flow）**：個體遷入或遷出，帶入或帶走等位基因
- **非隨機交配**：如同類交配，增加同型合子比例
