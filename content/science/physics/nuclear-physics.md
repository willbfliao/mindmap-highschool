# 原子核物理

## 原子核結構
### 核子
- 質子（proton）: 帶正電，質量 $\approx 1.673 \times 10^{-27}$ kg
- 中子（neutron）: 不帶電，質量略大於質子
- 核子（nucleon）: 質子與中子的統稱
### 原子序與質量數
- 原子序（$Z$）: 質子數，決定元素種類
- 質量數（$A$）: 質子數 + 中子數（$A = Z + N$）
- 同位素（isotope）: 質子數相同但中子數不同
  - 例: $^{1}\text{H}$、$^{2}\text{H}$（氘）、$^{3}\text{H}$（氚）
### 核力
- 強核力: 束縛核子的力，作用距離極短（$\approx 10^{-15}$ m）
- 不分質子或中子，同等吸引
- 短距離內遠大於庫侖排斥力
### 結合能
- 質量虧損（mass defect）: $\Delta m = Z m_{p} + N m_{n} - M_{\text{nucleus}}$
- 結合能: $E_{b} = \Delta m c^{2}$
- 平均結合能: $\frac{E_{b}}{A}$（每核子平均結合能）
- 鐵-56（$^{56}\text{Fe}$）平均結合能最大 → 最穩定

## 放射性衰變
### $\alpha$ 衰變
- $\alpha$ 粒子: $^{4}\text{He}$ 原子核（2 質子 + 2 中子）
- $^{A}_{Z} X \rightarrow ^{A-4}_{Z-2} Y + ^{4}_{2}\text{He}$
- 質量數減 4，原子序減 2
- 穿透力最弱（紙張即可擋住）
### $\beta$ 衰變
- $\beta^{-}$ 衰變: 中子 → 質子 + 電子 + 反微中子
  - $^{A}_{Z} X \rightarrow ^{A}_{Z+1} Y + e^{-} + \bar{\nu}$
  - 原子序加 1，質量數不變
- $\beta^{+}$ 衰變: 質子 → 中子 + 正子 + 微中子
  - $^{A}_{Z} X \rightarrow ^{A}_{Z-1} Y + e^{+} + \nu$
### $\gamma$ 衰變
- $\gamma$ 射線: 高能量電磁波（光子）
- 原子核從激發態躍遷至低能態時放出
- 原子序與質量數不變
- 穿透力最強（需厚鉛板或混凝土屏蔽）
### 半衰期
- 半衰期（half-life, $t_{1/2}$）: 放射性核種衰變一半所需時間
- $N(t) = N_{0} \times \left(\frac{1}{2}\right)^{t/t_{1/2}}$
- 與外在條件（溫度、壓力）無關
- 碳-14 定年法: $t_{1/2} = 5730$ 年，測定考古樣本年代

## 核分裂
### 核分裂反應
- 核分裂（nuclear fission）: 重原子核分裂為較輕的原子核
- $^{235}\text{U} + n \rightarrow ^{141}\text{Ba} + ^{92}\text{Kr} + 3n$（典型反應之一）
- 釋放大量能量（質量虧損轉為能量）
### 連鎖反應
- 每次分裂釋放 2~3 個中子 → 引發更多分裂
- 臨界質量（critical mass）: 維持連鎖反應的最少燃料量
- 控制棒: 吸收多餘中子以控制反應速率
### 核能發電
- 核反應爐: 控制連鎖反應的速率
- 冷卻劑: 帶走熱量產生蒸汽推動渦輪
- 優點: 不排放 $\text{CO}_{2}$、能量密度高
- 缺點: 核廢料處理、輻射安全風險

## 核融合
### 核融合反應
- 核融合（nuclear fusion）: 輕原子核結合為較重的原子核
- 太陽能量來源: $4 \, ^{1}\text{H} \rightarrow ^{4}\text{He} + 2e^{+} + 2\nu + \text{能量}$（質子-質子鏈）
- 條件: 需極高溫度（$> 10^{7}$ K）以克服庫侖排斥
### 核融合與核分裂的比較
- 核融合: 每核子釋放能量更多、燃料豐富（氫）、產物無放射性
- 目前挑戰: 電漿約束困難，尚無法穩定商業發電
- 氫彈: 不可控的核融合（以核分裂引爆）
