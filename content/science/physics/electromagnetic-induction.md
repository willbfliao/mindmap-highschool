# 電磁感應

## 法拉第電磁感應定律
### 磁通量
- 磁通量（magnetic flux）: $\Phi = BA \cos\theta$
  - $B$: 磁場強度，$A$: 面積，$\theta$: $\vec{B}$ 與面法線的夾角
- 單位: 韋伯（Wb）= $\text{T} \cdot \text{m}^{2}$
### 法拉第定律
- 感應電動勢（induced EMF）: $\varepsilon = -\frac{d\Phi}{dt} = -N\frac{\Delta\Phi}{\Delta t}$
  - $N$: 線圈匝數
- 磁通量變化越快 → 感應電動勢越大
- 產生感應電動勢的方式: 改變 $B$、改變 $A$、改變 $\theta$
### 動生電動勢
- 導棒在均勻磁場中移動（$\vec{B}$、導棒方向、$\vec{v}$ 兩兩互相垂直）:
  - $\varepsilon = BLv$
  - $L$: 導棒長度，$v$: 導棒速度
- 斜向運動: 須取速度、磁場在相應方向的分量
- 本質: 勞侖茲力將電荷推向導棒一端

## 冷次定律
### 定律內容
- 冷次定律（Lenz's law）: 感應電流方向使其產生的磁場抵抗磁通量的變化
- 磁通量增加 → 感應電流產生反向磁場
- 磁通量減少 → 感應電流產生同向磁場
### 本質
- 能量守恆的體現
- 感應電流總是「反抗」造成它的原因
### 判斷方法
- 先判斷原磁場方向
- 再判斷磁通量增減
- 感應磁場方向與原磁場增減相反
- 用右手定則由感應磁場方向推出電流方向

## 發電機與變壓器
### 交流發電機
- 線圈在磁場中旋轉 → 磁通量週期性變化 → 產生交流電
- 感應電動勢: $\varepsilon = NBA\omega \sin\omega t = \varepsilon_{0} \sin\omega t$
- 最大電動勢: $\varepsilon_{0} = NBA\omega$
- 有效值: $\varepsilon_{\text{rms}} = \frac{\varepsilon_{0}}{\sqrt{2}}$
### 直流發電機
- 加裝換向器（commutator）→ 輸出脈動直流電
### 變壓器
- 原理: 電磁感應（交流電通過初級線圈產生變化磁場）
- 電壓比: $\frac{V_{1}}{V_{2}} = \frac{N_{1}}{N_{2}}$
- 理想變壓器功率守恆: $V_{1}I_{1} = V_{2}I_{2}$
- 升壓器: $N_{2} > N_{1}$ → $V_{2} > V_{1}$
- 降壓器: $N_{2} < N_{1}$ → $V_{2} < V_{1}$
- 長距離輸電: 升壓降流以減少 $I^{2}R$ 損耗

## 電磁波
### 電磁波的產生
- 加速運動的電荷產生電磁波
- 交變電場產生磁場，交變磁場產生電場 → 自我傳播
- 馬克士威預測，赫茲實驗證實
### 電磁波的性質
- 橫波: 電場與磁場互相垂直，且都垂直於傳播方向
- 真空中速度: $c = \frac{1}{\sqrt{\mu_{0}\varepsilon_{0}}} = 3 \times 10^{8} \text{ m/s}$
- $c = f\lambda$
- 不需介質即可傳播
