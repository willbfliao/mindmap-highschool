# 植物體內的物質運輸

## 水分運輸

### 水分吸收
- 根毛細胞透過**滲透作用**吸收土壤中的水分
- 根毛細胞的溶質濃度 > 土壤溶液 → 水向根毛移動
- 水從根毛 → 皮層 → 內皮層 → 木質部

### 水分上升的動力 — 蒸散-內聚力-張力學說（Cohesion-Tension Theory）

#### 蒸散作用（transpiration）
- 水從葉片氣孔以**水蒸氣**形式散失
- 為水分上升的**主要驅動力**
- 蒸散產生**張力（tension / negative pressure）**，將水柱向上拉

#### 內聚力（cohesion）
- 水分子間的**氫鍵**產生內聚力
- 使水在木質部導管中形成連續的**水柱**
- 水柱不易斷裂

#### 附著力（adhesion）
- 水分子與導管壁之間的吸引力
- 協助維持水柱

#### 整合機制
- 蒸散作用在葉端產生**拉力** → 經由水柱傳導至根部
- 水柱依靠**內聚力**保持連續
- 水從土壤被「拉」上到數十公尺高的樹冠

### 根壓（root pressure）
- 根部主動吸收礦物質 → 木質部溶質濃度升高 → 水經滲透作用進入
- 產生向上的正壓力
- 可觀察到**溢泌現象（guttation）**：水滴從葉緣排出
- 根壓為**次要動力**，不足以將水推送至高大喬木頂端

### 影響蒸散速率的因素
- **光照**：光照使氣孔開啟 → 蒸散增加
- **溫度**：溫度升高 → 水蒸氣擴散加快 → 蒸散增加
- **濕度**：空氣濕度低 → 葉片內外水蒸氣濃度梯度大 → 蒸散增加
- **風速**：風帶走葉片周圍水蒸氣 → 維持濃度梯度 → 蒸散增加
- **氣孔開閉**：氣孔關閉時蒸散大幅降低

### 蒸散的功能
- 提供水分上升的**拉力**
- 促進**礦物質**從根到葉的運輸
- **散熱降溫**：蒸發需吸收熱量

## 礦物質的吸收與運輸

### 礦物質吸收
- 根部以**主動運輸**吸收土壤中的礦物質離子
- 需消耗 **ATP**
- 選擇性吸收：不同離子有不同的載體蛋白

### 必需礦物質元素
- **巨量元素**：N、P、K、Ca、Mg、S（需求量大）
- **微量元素**：Fe、Mn、Zn、Cu、B、Mo、Cl（需求量小但必不可少）
- 缺乏特定元素會出現特徵性症狀：
  - 缺 N：葉片黃化（老葉先）
  - 缺 Fe：葉脈間黃化（新葉先）
  - 缺 P：葉片呈紫色
  - 缺 K：葉緣焦枯

### 運輸途徑
- 礦物質溶於水中，隨木質部的水分向上運輸
- 部分礦物質可在韌皮部中再分配

## 有機養分的運輸（韌皮部運輸）

### 壓力流學說（Pressure-Flow Hypothesis / Mass Flow）

#### 基本概念
- 有機養分（主要為**蔗糖**）在韌皮部的**篩管**中運輸
- 從**源（source）**運送到**匯（sink）**
- **源**：產生或釋出有機物的器官（如行光合作用的成熟葉片）
- **匯**：消耗或儲存有機物的器官（如根、果實、生長中的嫩葉）

#### 機制
1. **裝載（loading）**：源端的伴細胞以**主動運輸**將蔗糖裝入篩管
2. 篩管內蔗糖濃度升高 → 水由鄰近木質部經**滲透作用**進入篩管
3. 篩管內**膨壓增加** → 產生壓力推動篩管液流動
4. **卸載（unloading）**：匯端蔗糖被卸載至周圍細胞
5. 篩管內溶質濃度降低 → 水離開篩管 → 回到木質部
6. 源端與匯端的**壓力差**驅動篩管液的整體流動

### 韌皮部運輸的特性
- 運輸方向可為**向下或向上**（取決於源匯位置）
- 運輸速度約 **50～100 cm/h**
- 運輸物質：蔗糖為主，也包含胺基酸、激素等
- 篩管為活細胞，運輸需要能量（裝載與卸載為主動運輸）

### 環剝實驗
- 去除一圈樹皮（含韌皮部）→ 環剝處上方隆起
- 證明有機物在韌皮部中**向下**運輸
- 環剝處以下的根部最終因缺乏養分而死亡

## 木質部與韌皮部比較

| 項目 | 木質部 | 韌皮部 |
|------|--------|--------|
| 運輸物質 | 水分、礦物質 | 有機養分（蔗糖） |
| 主要方向 | 向上（根 → 葉） | 源 → 匯（可雙向） |
| 主要細胞 | 導管、假導管（死細胞） | 篩管、伴細胞（活細胞） |
| 驅動力 | 蒸散張力 | 壓力流（膨壓差） |
| 是否需能量 | 蒸散為被動 | 裝載卸載需 ATP |
