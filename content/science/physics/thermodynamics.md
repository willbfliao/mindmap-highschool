# 熱學

## 熱容量與比熱
### 熱容量
- 熱容量（heat capacity）: C = Q / ΔT
- 物體升溫 1°C（或 1K）所需的熱量
- 單位: J/°C 或 J/K
### 比熱
- 比熱（specific heat）: c = Q / (mΔT)
- 每公斤物質升溫 1°C 所需的熱量
- 水的比熱: c = 4186 J/(kg·°C)，比大多數物質高
- Q = mcΔT: 熱量計算公式
### 熱量交換
- 熱平衡: 高溫物體放熱 = 低溫物體吸熱
- Q_放 = Q_吸 → m₁c₁(T₁ - T_f) = m₂c₂(T_f - T₂)
- 混合溫度: T_f = (m₁c₁T₁ + m₂c₂T₂) / (m₁c₁ + m₂c₂)

## 理想氣體方程式
### 氣體定律
- 波以耳定律（Boyle's law）: PV = 常數（T 固定）
- 查理定律（Charles's law）: V/T = 常數（P 固定）
- 給呂薩克定律（Gay-Lussac's law）: P/T = 常數（V 固定）
### 理想氣體方程式
- PV = nRT
- R: 氣體常數 = 8.314 J/(mol·K)
- n: 莫耳數
- T: 絕對溫度（K）
- PV = NkT（N 為分子數，k 為波茲曼常數）

## 氣體動力論
### 基本假設
- 氣體由大量分子組成，分子間距遠大於分子大小
- 分子做隨機運動，碰撞為彈性碰撞
- 分子間除碰撞外無交互作用
### 壓力的微觀解釋
- 氣體壓力源自分子碰撞容器壁
- P = ⅓ · (N/V) · m · v²_avg（v²_avg 為方均根速率的平方）
- PV = ⅔ · N · ½mv²_avg
### 溫度的微觀意義
- 平均動能: ½mv²_avg = ³⁄₂ kT
- 溫度反映分子平均動能
- 方均根速率: v_rms = √(3kT/m) = √(3RT/M)
  - T 越高 → 分子運動越快
  - M 越大 → 分子運動越慢

## 熱力學定律（選修補充）
### 熱力學第零定律
- 若 A 與 C 熱平衡，B 與 C 熱平衡，則 A 與 B 熱平衡
- 溫度計測量的理論基礎
### 熱力學第一定律
- ΔU = Q - W（內能變化 = 吸熱 - 對外做功）
- 能量守恆在熱力學中的表述
### 熱力學第二定律
- 克勞修斯表述: 熱不能自發從低溫傳向高溫
- 開爾文表述: 不可能從單一熱源取熱完全轉換為功
- 熵增原理: 孤立系統的熵只增不減
