# 電流的磁效應

## 載流導線的磁場
### 直線電流的磁場
- 安培右手定則: 拇指指向電流方向，四指彎曲方向為磁場方向
- 磁場大小: B = μ₀I / (2πr)
  - μ₀: 真空磁導率 = 4π × 10⁻⁷ T·m/A
  - r: 距導線的距離
- 磁場呈同心圓分佈，與距離成反比
### 圓形電流的磁場
- 圓心處磁場: B = μ₀I / (2R)（R: 線圈半徑）
- N 匝線圈: B = μ₀NI / (2R)
- 磁場方向: 右手定則（四指沿電流方向彎曲，拇指為磁場方向）
### 螺線管的磁場
- 內部磁場近似均勻: B = μ₀nI（n: 單位長度匝數 = N/L）
- 磁場方向: 右手定則判斷
- 外部磁場近似為零（理想螺線管）

## 安培定律
### 定律內容
- 安培定律（Ampère's law）: ∮B·dl = μ₀I_enc
- 沿封閉路徑的磁場環積分 = μ₀ × 路徑所圍電流
### 應用
- 長直導線: B = μ₀I/(2πr)
- 螺線管: B = μ₀nI
- 對稱性高的問題特別好用

## 帶電質點在磁場中的運動
### 勞侖茲力
- 勞侖茲力（Lorentz force）: F = qv × B（向量外積）
- 大小: F = qvB sinθ（θ 為 v 與 B 的夾角）
- 方向: 右手定則（v × B 方向，負電荷取反）
- 特性: 勞侖茲力不做功（永遠垂直於速度）
### 帶電粒子在均勻磁場中的運動
- v ⊥ B 時: 等速率圓周運動
  - 迴旋半徑: r = mv / (qB)
  - 迴旋週期: T = 2πm / (qB)（與速度無關）
  - 迴旋頻率: f = qB / (2πm)
- v 與 B 成任意角: 螺旋運動（螺旋線）

## 載流導線在磁場中的受力
### 安培力
- 安培力（Ampere force）: F = IL × B（向量外積）
- 大小: F = BIL sinθ
- 方向: 佛來明左手定則
### 兩平行載流導線間的作用力
- 同向電流: 互相吸引
- 反向電流: 互相排斥
- 力的大小: F/L = μ₀I₁I₂ / (2πd)

## 霍爾效應
- 載流導體在磁場中: 電荷偏向一側，產生橫向電壓
- 霍爾電壓: V_H = BId / (nqA)
- 應用: 測量磁場強度、判斷載子類型（電子或電洞）
