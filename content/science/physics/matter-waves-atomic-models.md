# 物質波與原子模型

## 拉塞福散射
### $\alpha$ 粒子散射實驗
- 拉塞福（Rutherford, 1911）: 以 $\alpha$ 粒子轟擊金箔
- 實驗結果:
  - 大部分 $\alpha$ 粒子直接穿過（原子大部分是空的）
  - 少數偏折（受到集中正電荷的排斥）
  - 極少數被彈回（正面撞擊高密度原子核）
### 原子核模型
- 原子中心有極小、極重、帶正電的原子核
- 原子核大小 $\approx 10^{-15}$ m，原子大小 $\approx 10^{-10}$ m
- 電子繞原子核運行（古典類比：如行星繞太陽）
  - 此類比在現代量子力學中已修正為軌域（電子機率雲）
### 模型的困難
- 古典電磁理論: 加速運動的電荷會輻射能量
- 電子應螺旋式墜入原子核 → 原子不穩定（與事實矛盾）
- 無法解釋原子光譜的不連續性

## 波耳氫原子模型
### 波耳假設（Bohr, 1913）
- 定態假設: 電子只能在特定軌道運行，不輻射能量
- 量子化條件: 角動量 $L = mvr = n\hbar$（$n = 1, 2, 3...$，$\hbar = \frac{h}{2\pi}$）
- 躍遷假設: 電子在能階間躍遷時放出或吸收光子，$hf = |E_{n} - E_{m}|$
### 氫原子能階
- 能量: $E_{n} = \frac{-13.6}{n^{2}}$ eV（$n = 1, 2, 3...$）
- 基態（ground state）: $n = 1$，$E_{1} = -13.6$ eV
- 游離能: $13.6$ eV（使電子脫離氫原子所需的最小能量）
- 軌道半徑: $r_{n} = n^{2} a_{0}$（$a_{0} = 0.053$ nm，波耳半徑）
### 光譜系列
- 來曼系列（Lyman）: 躍遷至 $n = 1$（紫外線）
- 巴耳末系列（Balmer）: 躍遷至 $n = 2$（可見光）
- 帕申系列（Paschen）: 躍遷至 $n = 3$（紅外線）
### 模型的限制
- 只能精確解釋氫原子光譜
- 無法解釋多電子原子
- 無法解釋譜線的精細結構

## 德布羅意物質波
### 物質波假說
- 德布羅意（de Broglie, 1924）: 所有物質都具有波動性
- 物質波波長: $\lambda = \frac{h}{p} = \frac{h}{mv}$
- 動能與波長: $\lambda = \frac{h}{\sqrt{2mKE}}$
### 波粒二象性
- 光: 干涉和繞射展現波動性，光電效應展現粒子性
- 物質: 電子繞射實驗證實電子的波動性
### 實驗驗證
- 戴維森-乾末實驗（Davisson-Germer, 1927）: 電子束照射鎳晶體產生繞射圖樣
- 電子繞射圖樣與 X 射線繞射類似
- 測量結果與 $\lambda = \frac{h}{mv}$ 一致
### 與波耳模型的關聯
- 波耳量子化條件: $2\pi r = n\lambda$（軌道周長 = 整數倍物質波波長）
- 電子形成駐波 → 只有特定軌道被允許
