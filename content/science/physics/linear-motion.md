# 直線運動

## 位移與路徑長
- 位移（displacement）: 起點到終點的直線距離，向量
- 路徑長（distance）: 實際行走路線長度，純量
- 位移大小 ≤ 路徑長

## 速度與速率
### 平均速度與瞬時速度
- 平均速度（average velocity）: v_avg = Δx / Δt（向量）
- 平均速率（average speed）: 路徑長 / 時間（純量）
- 瞬時速度（instantaneous velocity）: Δt → 0 時的平均速度
### x-t 圖與 v-t 圖
- x-t 圖斜率 = 速度
- v-t 圖斜率 = 加速度
- v-t 圖下面積 = 位移

## 加速度
- 加速度（acceleration）: a = Δv / Δt
- 正加速度: 速度增加（加速）
- 負加速度: 速度減少（減速）
- 單位: m/s²

## 等加速度運動
### 三大公式
- v = v₀ + at: 速度-時間關係
- x = v₀t + ½at²: 位移-時間關係
- v² = v₀² + 2ax: 速度-位移關係
### 補充公式
- 平均速度: v_avg = (v₀ + v) / 2（僅適用等加速度）
- 第 n 秒位移: xₙ = v₀ + a(n - ½)

## 自由落體
- 定義: 僅受重力作用、初速為零的運動
- 加速度: a = g ≈ 9.8 m/s²（向下為正）
- 公式: v = gt、h = ½gt²、v² = 2gh
- 與質量無關: 真空中羽毛與鐵球同時落地
- 伽利略比薩斜塔實驗: 推翻亞里斯多德重物先落地說

## 鉛直上拋
- 初速 v₀ 向上，加速度 g 向下
- 最高點: v = 0，上升時間 t = v₀ / g
- 最大高度: h = v₀² / (2g)
- 上升與下降時間相同（忽略空氣阻力）
- 回到拋出點時速率等於初速率
