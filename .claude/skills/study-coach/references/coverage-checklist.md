## 跨科連結地圖

以下為常出現的學測跨科整合方向：

| 連結 | 科目 A | 科目 B | 說明 |
|------|--------|--------|------|
| 能量守恆 | 物理（功與能）| 化學（熱化學）| 能量轉換與守恆定律 |
| 遺傳機率 | 生物（孟德爾遺傳）| 數學（機率）| 遺傳比計算 |
| 環境議題 | 地科（氣候變遷）| 公民（永續發展）| 時事素養題 |
| 濃度計算 | 化學（莫耳濃度）| 數學（比例）| 計算題型 |
| 工業革命 | 歷史（近代史）| 地理（都市化）| 社會科統整 |
| 人權發展 | 歷史（啟蒙運動）| 公民（人權保障）| 概念連結 |
| 經濟模型 | 公民（供需法則）| 數學（函數圖形）| 圖形判讀 |
| 文化交流 | 國文（文學流變）| 歷史（文化史）| 文史整合 |
